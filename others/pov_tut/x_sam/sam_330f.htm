Feuille de Trèfle

Objets:     "torus".
Méthodes: "#declare", "union{...}", "object{...}", "intersection{...}".
La Construction détaillée:
Le rayon petit de tous cercles est R0.
Le rayon des cercles intérieurs est Ri.
Le rayon du cercle extérieur est Ra.
Il résulte du plan ci-contre :
Ri (vert foncé) = Ra/2 + R0/2.
Cela on a besoin pour laisser passer le bague intérieur au-dessus de centre (flèche rouge). Seulement en ce cas on peut couper les partes au-dessus à gauche et à droite du bague avec un plan (où "box") tourné correspondant. Le plan à droite est montré (en jaune transparent) dans le plan ci-contre.
Le tore intérieur (orange) et un des plans pour couper (jaune).
 
// rayon du cercle extérieur:
#declare Ra = 3.00 -R0;
// rayon des cercles intérieurs;
#declare Ri  = Ra /2+ R0/2;

#declare Part =
intersection{
torus{ Ri,  R0
       rotate<90,0,0>
       translate<0,-Ra+Ri,0>}
plane { <0,1,0>,0 rotate<0,0, 30>}
plane { <0,1,0>,0 rotate<0,0,-30>}
box{<-Ri-R0,-Ra-R0,-R0>,
     <Ri+R0, 0,R0>}
}// ------- Ende von "Part" ------
union{   //-----------------------
torus{ Ra,  R0
       rotate<90,0,0>
      } // Ende des Torus  -------

object{ Part rotate<0,0,0*120>}
object{ Part rotate<0,0,1*120>}
object{ Part rotate<0,0,2*120>}
}// Ende der Union
//---------------------------- end
