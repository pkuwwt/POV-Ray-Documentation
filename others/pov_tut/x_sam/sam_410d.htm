Bikonvexe optische Linse

Ein Beispiel für die Verwendung von "intersection".
Objekte:   "sphere".
Methoden: "#declare", "intersection".
Klicken Sie hier für ein Beispiel!

Das Konstruktionsprinzip für bikonvexe optische Linsen

Das Gemeinsame ("intersection") zweier überlappenden Kugeln:

 //-------------------------- Linse --------  
 #declare R   = 6.0;   //Kugel-Radius
 #declare Over= 0.1;   //Kugel-Überlappung
 intersection{
  sphere{<0,0,0>,R  translate <0,0,-R+Over>}
  sphere{<0,0,0>,R  translate <0,0, R-Over>}
  texture{ T_Glass3 }
  interior{ I_Glass }
  translate < 0,1.2,0>}
 //---------------------------------- end --
