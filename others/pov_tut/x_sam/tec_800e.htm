Airplane
This shows how to make an airplane
with simple csg generated shapes
by following steps:

1. How to Make the Body of an Airplane (1)
Objects: sphere, box, cone.
Methods: declare, union, difference, intersection, macro.
2. How to Make the Body of an Airplane (2)
Objects: sphere, box, cone.
Methods: declare, union, difference, intersection, macro.
3. How to add Windows (1)
Objects: prism, cylinder, torus, sphere.
Methods: difference, intersection, union, declare, macro.
4. How to add Windows (2)
Objects: prism, cylinder, torus, sphere.
Methods: difference, intersection, union, declare, macro.
5. How to add Wings
Objects: cone, cylinder, torus, sphere.
Methods: difference, intersection, union, declare, macro.
6. How to add Propeller Engines
Objects: prism, cone, cylinder, torus, sphere, box.
Methods: difference, intersection, union, declare, macro.
