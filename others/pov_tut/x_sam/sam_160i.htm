Recinto

Oggetti: "box".
Metodi: "union", "layered textures", "macro".
Un file include per usare questo oggetto come un oggetto pronto per l'uso si trova su la mia paggina Oggetti POV-Ray.
La costruzione nel dettaglio:
Due motivi a righe, ognuno realizato con "color_map", consistente di una riga sottile nero e una riga larga chiara e trasparente, sono ruotati di 45 gradi anzi di -45 gradi. La sovrapposizione di questi motivi fanno un recinto di rete metallica. 
//------------------------ grid macro -----
#macro Raster(RScale, RLine)
pigment{
   gradient x scale RScale
   color_map{[0.000   color rgbt<0,0,0,0>]
             [0+RLine color rgbt<0,0,0,0>]
             [0+RLine color rgbt<1,1,1,1>]
             [1-RLine color rgbt<1,1,1,1>]
             [1-RLine color rgbt<0,0,0,0>]
             [1.000   color rgbt<0,0,0,0>]
            }}
 finish {diffuse 0.9}
#end// of Raster(RScale, RLine)-macro
//-----------------------------------------
#macro ClearGrid (Rscale )
texture{Raster(1.00,0.04) rotate<0 ,45,0>
        scale Rscale rotate<90,0,0>}
texture{Raster(1.00,0.04) rotate<0,-45,0>
        scale Rscale rotate<90,0,0>}
#end //
//-----------------------------------------
box {<0,0,0>,<2,1.99,0.001>
     ClearGrid (0.2)}
//---------------------------------- end --
