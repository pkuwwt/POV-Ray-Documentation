Lente ottica biconvessa

La costruzione della forma di una lente biconvessa.
Un esempio per l'uso di "intersection".
Oggetti: "  "sphere".
Metodi:  "#declare", "intersection".
Cliccare qui per un esempio!
I principi della costruzione di una lente biconvessa.

Le parti in comune (= "intersection")
di due sfere che si sovrappostano parzialmente:

 //------------------------  lente --------  
 #declare R   = 6.0;   //sphere radius
 #declare Over= 0.1;   //sphere overlap
 intersection{
  sphere{<0,0,0>,R  translate <0,0,-R+Over>}
  sphere{<0,0,0>,R  translate <0,0, R-Over>}
  texture{ T_Glass3 }
  interior{ I_Glass }
  translate < 0,1.2,0>}
 //---------------------------------- end --
