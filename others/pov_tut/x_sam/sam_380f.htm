Octogone - Prisme Octogonal

La construction de un octogone (écrou ou colonne octogonal)
Exemple pour l'usage du commande "intersection".
Objets:     "box", "cylinder".
Méthodes: "intersection", "inverse"
Click here for an example!

La construction :

Les parties communes (= intersection) d'une forme quadratique et la même forme pivotée autour l'axe y par 45 degrés forment un objet avec une silhouette octogonale.

Plongée (regardé vers la direction y)

Le text: 

//================== Octogone ===============
intersection{
  box {<-1, 0.0,-1>,< 1,1.0,1> rotate<0, 0,0>}
  box {<-1,-0.1,-1>,< 1,1.1,1> rotate<0,45,0>}
  scale <1,0.5,1>
  texture{Polished_Brass
          normal{bumps 0.3 scale 0.02}
          finish{diffuse 0.9 phong 1}
         }
  translate<1.1,0.00,0>}
//============================================


//=============== Octogone creusé ===========
intersection{
  box {<-1, 0.0,-1>,< 1,1.0,1> rotate<0, 0,0>}
  box {<-1,-0.1,-1>,< 1,1.1,1> rotate<0,45,0>}
  cylinder{<0,-0.1,0>,<0,1.1,0>,0.5 inverse
  scale <1,0.5,1>
  texture{Polished_Chrome
          normal{bumps 0.3 scale 0.2}
          finish{diffuse 0.9 phong 1}
         }
  translate<-1.1,0.50,0.5>}
//---------------------------------------- end
