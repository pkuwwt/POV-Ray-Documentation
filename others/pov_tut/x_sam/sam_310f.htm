Le calcul du rayon Ri
des cercles intérieurs :
Il résulte du plan ci-contre :
Ri (orange) = 1/2*Ra.
Si le centre du cercle extérieur est à
M0 = <0,0,0>,
les centres des cercles intérieurs sont à
M1 = <Ri,0,0> et M1 = <-Ri,0,0> .

Dans ce qui suit s'applique pour le segment de tore mon macro "#macro Segment_of_Torus( R_major, R_minor, Segment_Angle)" de mon fichier inlude.
Pour cela on doit intégrer dans le texte le fichier correspondant avec #include "shapes3.inc" .
Le calcul du rayon Ri
des cercles intérieurs.
 
// center:
#declare M  = <0,0,0>;
// outer radius:
#declare Ra = 3.00 -R0;
// radius of inner half circles:
#declare Ri  = Ra /2 ;

//  2 torus segments
#include "shapes3.inc"

union{   //-----------------------
torus { Ra,  R0
        rotate<90,0,0>
        translate M0
  } // end of torus  -------------
object{
   Segment_of_Torus( Ri, R0, -180)
   rotate<-90,0,0>
   translate < Ri,0,0>
 } // end of Torus_Segment(...) --
object{
   Segment_of_Torus( Ri, R0, -180)
   rotate<-90,0,0> rotate<0,0,180>
   translate <-Ri,0,0>
 } // end of Torus_Segment(...) --
}// end of union
//---------------------------- end
