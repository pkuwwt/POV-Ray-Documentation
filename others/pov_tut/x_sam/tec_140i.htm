Camion
4. Trattore con Rimorchio

Oggetti:    "Round_Box", "cylinder".
Metodi: "difference", "intersection", "union", "#declare" "#macro".
Un file include per usare questo oggetto come un oggetto pronto per l'uso si trova su la mia paggina Oggetti POV-Ray.
Costruzione di un Rimorchio:
Il rimorchio si mette insieme di un "Round_Box" e un asse doppio come a l' trattore.

Il Camion completo: Trattore + Rimorchio.

L'assemblaggio si fa con oggetto "macro", per lassiare la scelta libera l'angolo del volante, l'angolo del rimorchio e anche la verniciatura della carrozzeria. 
//------------------------------------
#declare Trailer_Angle    =0;//~+/-110
#declare Front_Wheel_Angle=0;//~ +/-60
#declare Truck_Texture =
   texture{ pigment{ color rgb<1,1,1>}
            finish{ diffuse .9 phong 1}
          } // end of texture
//------------------------------------
object{ Truck0( Front_Wheel_Angle,
                Trailer_Angle,
                Truck_Texture
              ) //--------------------
rotate<0,0,0>
translate<4.20,0,0>
} //----------------------------------

 
 
 
Cliccare qui per una descrizione della scena per POV-Ray:
"truck0_1.txt" o "truck0_1.pov"
