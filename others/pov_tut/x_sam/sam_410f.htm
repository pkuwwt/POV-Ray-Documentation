Lentille optique biconvexe

La construction de la forme d'une lentille biconvexe.
Un exemple pour l'usage de "intersection".
Objets:   "sphere".
Méthodes: "#declare", "intersection".
Cliquez ici pour un exemple !
Les principes da la construction d'une lentille biconvexe.

Les parties communes (= intersection) de deux sphères que se chevauchant partiellement :

 //--------------------------  lens --------  
 #declare R   = 6.0;   //sphere radius
 #declare Over= 0.1;   //sphere overlap
 intersection{
  sphere{<0,0,0>,R  translate <0,0,-R+Over>}
  sphere{<0,0,0>,R  translate <0,0, R-Over>}
  texture{ T_Glass3 }
  interior{ I_Glass }
  translate < 0,1.2,0>}
 //---------------------------------- end --  
 
La description de cette scène
pour POV-Ray :

"povlens.txt" ou "povlens.pov"
