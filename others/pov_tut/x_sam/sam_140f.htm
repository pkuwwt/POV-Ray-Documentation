Fenêtre

Objets:   "box"
Méthodes: "difference", "union", "#declare"
La Constuction dans tous les détails:
Le châssis de fenêtre se forme par une difference d'une boîte mince ("box" = parallélépipède rectangle) aux dimensions extérieur de la fenêtre d'où sont soustraite quelque boîte mince aux dimensions des trous pour les vitres. Le résultat de cette difference reçoit le couleure du châssis de fenêtre. À la fin on ajoute un autre boîte très mince avec la texture de verre avec le mot clé "union".
Il ne faut pas utiliser un matériau de verre ("material{M_Glass}" avec "ior" et réfraction réelle) quand nous voulons seulement un simple verre à vitres. Utiliser "glass material" seulement ralente la vitesse du calcul d'image - avec un verre très minice réfraction ne fait aucun effet visible!

Un ficher include pour utiliser cet objet comme un objet prêt à l'usage vous pouvez trouver sur mon site Objets POV-Ray.
Il Piano :
vue frontale (en direction z)
 
//---------------- Finestra ----------------
#declare Window_Texture =
  texture{ pigment{ color White}
           finish { ambient 0.9 phong 0.5}
         } // end of texture
//------------------------------------------
#declare Window =
union{
 difference{
  box{<-0.50,0.80,-0.02>,< 0.50,1.80,0.02>}
  box{<-0.45,0.85,-0.03>,<-0.03,1.27,0.03>}
  box{< 0.03,0.85,-0.03>,< 0.45,1.27,0.03>}
  box{<-0.45,1.33,-0.03>,<-0.03,1.75,0.03>}
  box{< 0.03,1.33,-0.03>,< 0.45,1.75,0.03>}
  texture{ Window_Texture }
  } // --- end of difference
 box{<-0.49,0.81,0.0>,< 0.49,1.79,0.001>
     texture{ T_Glass3 } // without ior !!!
     } // end of box
 } // --- end of union
// Attenzione: evitare superficie coincidenti!!
//--- fine di definizione della finestra ------

// Finestra:
object{Window translate<0,0,0>}
//------------------------------------- end
Les 'box' jaune sont soustrait du 'box' blanc !
