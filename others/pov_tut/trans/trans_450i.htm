Lo Scorrimento Elastico con matrix

Non é raccomandabile di definire con matrices le trasformazioni semplici comme spostamento (translate), stiracchiare (scale) o rotazione (rotate) perché questo é und modo molto dispendioso e difficile à capire.
Lo scorrimento elastico é una trasformazione descrivibile solo con matrice in POV-Ray

Esempio: scorrimento elastico dell'asse y verso la direzione x con il fattore 0.5 (30°):


  matrix< 1 , 0, 0,
         0.5, 1, 0,
          0 , 0, 1,
          0 , 0, 0 >


Le matrices delo scorrimento elastico visualizzate con un cubo:

matrix < 1,0.5,  0,
         0,  1,  0,
         0,  0,  1,
         0,  0,  0 >
  
matrix < 1,  0,0.5,
         0,  1,  0,
         0,  0,  1,
         0,  0,  0 > 
matrix < 1,  0,  0,
       0.5,  1,  0,
         0,  0,  1,
         0,  0,  0 >
  
matrix < 1,  0,  0,
         0,  1,0.5,
         0,  0,  1,
         0,  0,  0 > 
matrix < 1,  0,  0,
         0,  1,  0,
       0.5,  0,  1,
         0,  0,  0 >
  
matrix < 1,  0,  0,
         0,  1,  0,
         0,0.5,  1,
         0,  0,  0 >
Per le matrices delo scorrimento elastico si può usare dei modelli di testo della sezione "Shearing & transform" del mio "Insert Menu Add-on".
