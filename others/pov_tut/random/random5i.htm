Piantare alberi con casualità      

Come si può piantare una serie d'alberi in modo semi-irregolare.
Pre i dettagli come si fa quest'alberi, si può vedere il file de la scena.

Se piantiamo alberi con un while loop involuto il resultato è molto non naturale perché toppo regolare. Questo posso essere il metodo ottimo per tagliare gli alberi per fare legna da ardere, ma non ha un bell'aspetto!


 
 
 
 
 
 

Cambiare la scala per caso,
inclinare e girare per caso: 

//------------------------------------
 // tilting around z direction
 rotate<10*2*(0.5-rand(Random_3)),0,0>
 // turning around y axis
 rotate<0,360*rand(Random_0),0>
 // changing scale
 scale  1+0.5*rand(Random_1)
///-----------------------------------

Movimento casuale: 

//------------------------------------
 // shifting in x and z:
 translate
  < NrX*2.75+2*(-0.5+rand(Random_2)),
    0,
    NrZ*2.75+2*(-0.5+rand(Random_2))>}
///-----------------------------------
