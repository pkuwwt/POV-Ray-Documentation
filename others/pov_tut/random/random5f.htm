 Les arbres avec hazard.

Comment planter une série des arbres semi-irrégulière avec un boucle double.
Pour plus de détails comme on fait ces arbres utilisés ici, voir le fichier correspondant de la scène.

Si nous plantons les arbres avec un boucle double le résultat est d'une régularité très artificielle.
Cela peut être très practique pour abattre ces arbres pour faire de bois de chauffage, mais cela n'est pas belle à voir !


 
 
 
 
 
 

Les dimensions aléatoires ('size'),
incliner par hasard et
pivoter par hasard ('rotate') : 

//------------------------------------
 // tilting around z direction
 rotate<10*2*(0.5-rand(Random_3)),0,0>
 // turning around y axis
 rotate<0,360*rand(Random_0),0>
 // changing scale
 scale  1+0.5*rand(Random_1)
///-----------------------------------

Mouvement aléatoire : 

//------------------------------------
 // shifting in x and z:
 translate
  < NrX*2.75+2*(-0.5+rand(Random_2)),
    0,
    NrZ*2.75+2*(-0.5+rand(Random_2))>}
///-----------------------------------
