Combinare coppie d'immagini stereo
Si puo combinare una coppia d'immagini stereo usando 'image_map' una fotocamera 'orthographic'.

Supponendo di avere
una coppia di immagini stereo:
Vista dell'occhio destro:  Stereo_Image_1.jpg
Vista dell'occhio sinistro: Stereo_Image_2.jpg

Nota: Nella seguente scena POV-Ray
non c'è una 'light_source'.
Le uniche cose visibili nella scena sono
le immagini, con 'finish{ ambient 1 }'.

L'esempio seguente funziona con POV-Ray 3.7,
per POV-Ray 3.6.2, dobbiamo aggiungere una
correzione gamma delle immagini usate
da una macro. Per maggiori dettagli si veda
il file di scena allegato.

Se le immagini hanno la dimensione 240 x 240
pixel, abbiamo bisogno della seguente riga di
comando: (immagine qui in basso)
Con POV-Ray 3.7 e output jpeg:
   +w480 +h240 +fj100 .
Con POV-Ray 3.6.2 usiamo:
   +w480 +h240
   e dobbiamo sostituire
   le estensioni ".jpg" con ".bmp"
   e il tipo di immagine 'jpeg' con 'sys',
   nelle image_maps qui sotto. 
//--------------------------------
camera{
 orthographic
 location <0, 0.5,-8.18>
 right x*image_width/image_height
 angle 14
 look_at  <0, 0.5, 0.00>
} //------------------------------
box{ <0,0,0>,< 1, 1, 0.01>
     texture{
       pigment{
         image_map{
       jpeg "Stereo_Image_1.jpg"
         map_type 0
         interpolate 2
         once
         } // end of image_map
       } //  end of pigment
       finish{ ambient 1 }
    } // end of texture
    translate<-1,0,0>
} // end of box //----------------
//--------------------------------
box{ <0,0,0>,< 1, 1, 0.01>
     texture{
       pigment{
         image_map{
       jpeg "Stereo_Image_2.jpg"
         map_type 0
         interpolate 2
         once
         } // end of image_map
       } //  end of pigment
       finish{ ambient 1 }
    } // end of texture
    translate< 0,0,0>
} // end of box //----------------
(Per maggiori dettagli il file di scena adiacente !!! )
