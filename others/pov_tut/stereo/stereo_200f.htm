En combinant les paires des images stéréo.
La jonction des paires des images stéréo
on peut faire avec POV-Ray en utillisant
'image_map' et caméra 'orthographic'.

   En supposant que nous avons
une paire d'images stéréo:
Vue oeil droit:     Stereo_Image_1.jpg
Vue oeil gauche: Stereo_Image_2.jpg

Remarque: Dans la scène POV-Ray suivante
il n'ya pas un source de lumière !
Les seules choses visibles en scène
sont les images, avec ' finish{ambient 1} '.

L'exemple suivant fonctionne avec POV-Ray 3.7.
Pour POV-Ray 3.6.2, nous devons ajouter
une correction gamma des images d'entrée par
une macro. Au-delà nous avons besoin de
remplacer les extensions ".jpg" par ".bmp"
et le type d'image 'jpeg' par 'sys', dans les
image_maps ci-dessous. Pour plus de détails,
voir le fichier de scène ci-joint.

Si les images ont la taille 240 x 240 pixels,
nous avons besoin d'une ligne de commande suivante:
   With POV-Ray 3.7 et de sortie jpeg:
+w480 +h240 +fj100 .
   With POV-Ray 3.6.2 nous devons utiliser:
+w480 +h240 . 
//--------------------------------
camera{
 orthographic
 location <0, 0.5,-8.18>
 right x*image_width/image_height
 angle 14
 look_at  <0, 0.5, 0.00>
} //------------------------------
box{ <0,0,0>,< 1, 1, 0.01>
     texture{
       pigment{
         image_map{
       jpeg "Stereo_Image_1.jpg"
         map_type 0
         interpolate 2
         once
         } // end of image_map
       } //  end of pigment
       finish{ ambient 1 }
    } // end of texture
    translate<-1,0,0>
} // end of box //----------------
//--------------------------------
box{ <0,0,0>,< 1, 1, 0.01>
     texture{
       pigment{
         image_map{
       jpeg "Stereo_Image_2.jpg"
         map_type 0
         interpolate 2
         once
         } // end of image_map
       } //  end of pigment
       finish{ ambient 1 }
    } // end of texture
    translate< 0,0,0>
} // end of box //----------------

Stereo_Image_1.jpg : la vue oeil droit.
Stereo_Image_2.jpg : la vue oeil gauche.

L'image stéréo combiné pour la vision croisée.

  La linge de commande pour Stereo_Image_Combine_.pov.

POV-Ray 3.7 fichier de la scène :
Stereo_Image_Combine_.pov
POV-Ray 3.6 fichier de la scène :
Stereo_Image_Combine_36.pov

Nous devons mettre ce fichier POV-Ray dans le répertoire de notre
images - Dans le cas contraire, nous devons ajouter le chemin complet
pour les noms de fichier d'image dans le texte de la scène.
