Cubo e
Cuboide
Qualche fondamento geometrico
molto utile
sul esaedro regolare e
parallelepipedo rettangolo

Nel seguente si scriva per la radice quadrata di un numero, l'espressione inglese "sqrt(Numero)"
in conformità alla sintassi usata con POV-Ray.

Dimensioni del cubo
Lunghezza degli spigoli di un cubo: a.
Lunghezza d dei diagonali di facce di un cubo:
    d = sqrt (2) * a ;
Lunghezza D dei diagonali a traverso di un cubo:
    d = sqrt (3) * a ;
Il raggio della circumsfera:
    R = 1/2*sqrt( 3 ) * a ;
Dimensioni del cuboide
Lunghezza degli spigoli: a, b, c.
Lunghezza D dei diagonali a traverso di un cuboide:
d = sqrt ( a2 + b2 + c2 )
