Le Théorème de Pythagore Quelques qualités géométriques du triangle rectangle.
Note: En POV-Ray nous utilisons "sqrt(X)" pour la racine carrée de X et nous utilisons "X*X" ou "pow(X,2)" pour X2.
Dans un triangle rectangle
avec la hypoténuse c, et les cathètes a et b
nous avons le théorème de Pythagore :
    a2 + b2 = c2
( Dans un triangle rectangle, le carré de la longueur de l’hypoténuse (côté opposé à l’angle droit) est égal à la somme des carrés des longueurs des côtés de l’angle droit. )
Le calcul des côtés:

 
   c = sqrt( a*a + b*b ) ; ou
   c = sqrt( pow( a , 2) + pow( b , 2 ) ) .

 
   a = sqrt( c*c - b*b ) ;

 
   b = sqrt( c*c - a*a ) ;
 
Les cathètes a et b :
  a2 = p*c   or
  b2 = q*c
L'hauteur hc du triangle rectangle:
  h2 = p * q
