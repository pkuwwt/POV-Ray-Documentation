Cube et
Cuboïde
Quelques qualités
géométriques

Note: En POV-Ray nous utilisons "sqrt(X)" pour la racine carrée de X
et nous utilisons "X*X" ou "pow(X,2)" pour X2.

Dimensions du cube
Longueur d'un côté du cube : a.
Longueur d'une diagonale d'un face du cube :
    d = sqrt (2) * a ;
Longueur d'une diagonale à travers d'un cube :
    d = sqrt (3) * a ;
Le rayon de la sphère circonscrite :
    R = 1/2*sqrt( 3 ) * a ;
Dimensions d'un cuboïde
Longueurs des côtés d'un cuboïde : a, b, c.
Longueur d'une diagonale à travers d'un cuboïde :
d = sqrt ( a2 + b2 + c2 )
