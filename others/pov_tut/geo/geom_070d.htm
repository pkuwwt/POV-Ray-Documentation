Der Satz des Pythagoras Einige praktische geometrischen Eigenschaften des rechtwinkligen Dreiecks.
Anmerkung: In POV-Ray verwendet man "sqrt(X)" für die Qadratwurzel von X und "X*X" oder "pow(X,2)" für X2.
In einem rechtwinkligen Dreieck mit der Hypotenuse c,
und den Katheten a und b,
besagt der Satz des Pythagoras :
    a2 + b2 = c2
(Die Summe der Quadrate über den Katheten ist gleich dem Quadrat über der Hypotenuse.)

Berechnung der Seiten:

 
   c = sqrt( a*a + b*b ) ; oder
   c = sqrt( pow( a , 2) + pow( b , 2 ) ) .

 
   a = sqrt( c*c - b*b ) ;

 
   b = sqrt( c*c - a*a ) ;

Kathetensatz:
  a2 = p*c   oder
  b2 = q*c
Höhensatz:
  h2 = p * q
    Der Satz des Pythagoras am rechtwinkligen Dreieck
  Höhensatz: h2 = p * q
