Würfel und
Quader
Einige interessante
geometrische
Eigenschaften
Im folgenden wird für die Quadratwurzel einer Zahl, der englische Ausdruck "sqrt(ZAHL)"
gemäß der von POV-Ray verwendeten Syntax geschrieben.
 

Dimensionen des Würfels
Länge der Würfel-Kante: a.
Länge der Seiten-Diagonalen:
    d = sqrt (2) * a ;
Länge der Raum-Diagonalen:
    d = sqrt (3) * a ;
Der Radius der Umkugel:
    R = 1/2*sqrt( 3 ) * a ;
Dimensionen des Quaders
Länge der Kanten des Quaders: a, b, c.
Länge einer Raumdiagonalen beim Quader:
d = sqrt ( a2 + b2 + c2 )
