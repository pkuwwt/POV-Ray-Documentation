Dodecaedro Regolare
Qualche fondamento geometrico molto utile.

Nel seguente si scriva per la radice quadrata di un numero, l'espressione inglese "sqrt(Numero)"
in conformità alla sintassi usata con POV-Ray.
 

Dimensioni del dodecaedro regolare
Lunghezza degli spigoli di un dodecaedro: a.
Il raggio della circumsfera:
R = a / 4 * sqrt( 3 ) * ( 1 + sqrt(5) ) ;

Il raggio de la sfera tangente gli spigoli:
Re = a / 4 * ( 3 + sqrt(5) ) ;

Il raggio della sfera inscritta:
Ri = a / 2 * sqrt( ( 25 + 10*sqrt(5))/10 );

L'angolo entro due facce, Angolo diedrale: ~116.57°
Face_Angle =
degrees( acos(-1/5*sqrt(5)));

L'angolo entro due spigoli: ~121.72°
Face_Edge_Angle =
degrees( acos( -sqrt( (5-sqrt(5))/10 ) ) );
