Cube and
Cuboid
Some useful
geometrical facts.
In the following we write for the square root of a number the expression "sqrt(ZAHL)"
conforming to the syntax used in POV-Ray.
 

Dimensions of a cube
Length of an edge of the cube: a.
The length of the diagonals of the faces:
    d = sqrt (2) * a ;
The length of the diagonals through a cube:
    d = sqrt (3) * a ;
The radius of circumsphere:
    R = 1/2*sqrt( 3 ) * a ;
Dimensions of a cuboid
Length of the edges of the cubid: a, b, c.
The length of a diagonal through a cuboid:
d = sqrt ( a2 + b2 + c2 )
