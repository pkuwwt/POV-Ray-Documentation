Wasser in einem Glas
Zur Simulation von Wasser im Glas benötigen wir ein transparentes Material mit einem Brechungsindex von ior = 1.33.
material{
  texture{
    pigment{ rgbf<.93,.95,.98,0.825>*0.99}
    finish { ambient 0.0 diffuse 0.15
             reflection{0.1,0.1 fresnel_on}
             specular 0.6 roughness 0.005
             conserve_energy
           } // end finish
  } // end of texture

  interior{ ior 1.33
             fade_power 1001
             fade_distance 0.5
             fade_color <0.8,0.8,0.8>
             caustics 0.16
   } // end of interior
 } // end of material
//-----------------------------------------
Ein Material für Wasser
Um diese Form mit einer welligen Oberseite zu erhalten bilden wir die 'intersection' (Schnittmenge) des obigen Körpers mit demselben, nur etwas weiter und ein klein wenig niederer mit demselben Material ergänzt ein 'normal' in der 'texture': 
     normal{ ripples 1.15
             scale 0.0125
             turbulence 0.3
             translate<-0.05,0,0>
             rotate<0,-20,0>
           } // end normal

//-----------------------------------------
Ein Material für Wasser mit welliger Oberseite
Alles was man jetzt noch tun muss: Man fügt ein einfaches Glas hinzu!
(Das Glas-Material hat einen höheren IOR (~1.5) und eine etwas andere Farbe).
