La mer agitée,
la surface de l'eau plus agité :
 Nous obtenons cela par ajouter à la texture le commande suivante : 
normal{ bumps 0.3
        scale 0.05
        turbulence 0.6
      }
//-----------------------------------







Aussi un effet très realiste on peut voir par ex. comme cela : 
normal{ crackle 0.15
        scale 0.75
        turbulence 0.3
        translate<-1,0,5>
      }
//-----------------------------------
La surface de l'eau plus agité avec 'bumps'. Description de cette scène pour POV-Ray : p_watr2.txt
La surface de l'eau plus agité avec 'crackle'. Description de cette scène pour POV-Ray : p_watr3.txt

Une surface de l'eau
ridée par le vent :
 Nous obtenons cela par le motif 'crackle' ou 'bozo' et par ajouter des échelles differentes en direction x et y et aussi une rotation dans la direction transversal au vent :
 
normal{ bozo 1.75
         scale <2.0,1,0.3>*0.20
         rotate<0,10,0>
         turbulence 0.9
         translate<0,0,-2>
       }
//-------------------------------------
La surface de l'eau agité par le vent.
Description de cette scène pour POV-Ray : p_watr4.txt
L'effet ressemblent avec 'crackle': 
normal{ pigment_pattern{
             crackle
             colour_map { [0, rgb 0]
                          [1, rgb 1] }
             turbulence 1.7
             translate <0,0,1>}
             0.5}
       scale <2.0,1,0.3>*2
       rotate<0,10,0>
     }
//-------------------------------------
Une surface de l'eau agité par le vent.
Description de cette scène pour POV-Ray : p_watr4b.txt
Remarque : Tout cela sont seulement des simulations des surfaces ondulées! Géométriquement, ils sont toujours parfaitement plat. Pour obtenir des des surfaces tridimensionnelles ondulées, il faut une beaucoup plus ! Voir ici et ici !
