Un ciel parsemé détoiles:

Pour créer un ciel de nuit avec des millions d'étoiles nous pouvons utiliser les textures des ciels parsemés détoiles fait de Jeff Burton avec les nomes "Starfield1, Starfield2, ..., Starfield6", que nous trouvons dans le fichier include standard "stars.inc. Nous devons utiliser ces textures avec une échelle appropriée (cela dépende un peu des dimensions de l'image). 

// starry sky ----------------
#include "stars.inc"
sphere{ <0,0,0>, 1
        texture{ Starfield1 scale 0.25
               } // end of texture
        scale 10000
      } //end of sphere ---------------
Un ciel parsemé détoiles avec la texture Starfield1.
