Cielo stellato:

Per creare un cielo stellato con milioni di picole stelle possiamo usare we le testure di cieli stellati fatto di Jeff Burton chiamate "Starfield1, Starfield2, ..., Starfield6", che possiamo trovare nello standard include file "stars.inc. Dobbiamo usarle con una scala adatta (questo dipende un poco delle dimensioni dell'immagine!). 

// starry sky ----------------
#include "stars.inc"
sphere{ <0,0,0>, 1
        texture{ Starfield1 scale 0.25
               } // end of texture
        scale 10000
      } //end of sphere ---------------
