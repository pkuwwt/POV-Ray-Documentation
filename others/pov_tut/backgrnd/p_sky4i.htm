Nuvole con piani impilati:

Qui molti piani sono impilati l'uno sopra l'altro e colorati con un motivo semitrasparente di nuvole che si svaniscono verso gli strati superiori e verso gli strati di base.
L'idea originale idea per questa tecnica ho trovato qui
Stacked Sky Planes macro "stsky.inc" di Jaime Vives Piqueres, 2000 a http://www.ignorancia.org/ q.v. "techniques" >> "Stacked Planes".

Ho fatto un include file adeguato per questo scoppo:
Stacked_Planes_Clouds_1.inc
Nota: Questo include file fa solo le nuvole. Dobbiamo aggiungere una sfera blue azzuro nel fondo (p.e. con sky_sphere{ ... } ).
Il handicap di questa technica: guardato del livello delle nuvole o vicino l'orizzonte si vede le breccie tra i piani.
