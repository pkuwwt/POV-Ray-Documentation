Himmel mit 2 Wolkenschichten

Dafür werden 3 Ebenen übereinander plaziert. Das höchste Level: Eine Ebene mit dem Blau des Himmels. Mit "finish{ ambient 1 diffuse 0)" vermeidet man Schatten auf dieser blauen Himmelsebene.
Darunter eine zweite Ebene mit einer teilweise transparenten Wolkenstruktur. Diese Ebene bildet die hochfliegenen Wolken. Darunter eine dritte Ebene mit einer teilweise transparenten Wolkenstruktur. Diese Wolken haben meist eine andere Bewegungsrichtung, was leicht durch ein Skalieren dieser Texturschicht simuliert werden kann.
Dies ist eine hellere Version von Darin Dugger's großartigen "T_Clouds" aus der include-Datei "e;skies.inc"e;. Sie wurde auch modifiziert um eine Verwendung von "fog" zu ermöglichen - dies geht nicht mit dem originalen Objekt "O_T_Cloud2"
Ein Himmel mit 2 Wolkenschichten.
