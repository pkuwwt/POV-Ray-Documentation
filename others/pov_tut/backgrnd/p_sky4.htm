Stacked planes clouds :

Here many planes were stacked on over the other and colored by a semitransparent pattern of clouds fading out at the top an at the base. The original idea for this technique I found at the Stacked Sky Planes macro "stsky.inc" by Jaime Vives Piqueres, 2000 at http://www.ignorancia.org/ see "techniques" >> "Stacked Planes".

I made an according include file for this
Stacked_Planes_Clouds_1.inc
Note: This include file produces only the clouds. You have to put a blue sky in the background (i.e. by sky_sphere{ ... } ).
Handicap of this technique: viewed for the cloud level or near the horizon we see the gaps between the plaines.
