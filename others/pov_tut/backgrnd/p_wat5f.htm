L'eau dans un verre
Pour simuler l'eau dans un verre, nous avons besoins d'un matière transparent avec un index de réfraction ior = 1.33.
material{
  texture{
    pigment{ rgbf<.93,.95,.98,0.825>*0.99}
    finish { ambient 0.0 diffuse 0.15
             reflection{0.1,0.1 fresnel_on}
             specular 0.6 roughness 0.005
             conserve_energy
           } // end finish
  } // end of texture

  interior{ ior 1.33
             fade_power 1001
             fade_distance 0.5
             fade_color <0.8,0.8,0.8>
             caustics 0.16
   } // end of interior
 } // end of material
//-----------------------------------------
Un matière transparent pour l'eau.
Pour recevoir cette forme avec une coté supérieur onduleux, nous faisons une 'intersection' de cette forme avec cette forme un peu plus large e un peu plus profond avec ce matière avec un 'normal' come suivante dans la texture: 
     normal{ ripples 1.15
             scale 0.0125
             turbulence 0.3
             translate<-0.05,0,0>
             rotate<0,-20,0>
           } // end normal
//-----------------------------------------
Un matière transparent pour l'eau avec une coté supérieur onduleux.
