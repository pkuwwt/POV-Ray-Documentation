Acqua in un Bicchiere.
Per simulare l'acqua in un bicchiere, abbiamo bisogno di un materiale trasparente con un indice di rifrazione ior = 1.33.
material{
  texture{
    pigment{ rgbf<.93,.95,.98,0.825>*0.99}
    finish { ambient 0.0 diffuse 0.15
             reflection{0.1,0.1 fresnel_on}
             specular 0.6 roughness 0.005
             conserve_energy
           } // end finish
  } // end of texture

  interior{ ior 1.33
             fade_power 1001
             fade_distance 0.5
             fade_color <0.8,0.8,0.8>
             caustics 0.16
   } // end of interior
 } // end of material
//-----------------------------------------
Un materiale per l'acqua
Per ottenere questa forma con un lato superiore ondulato, facciamo una 'intersection' del corpo qui in alto con lo stesso corpo, ma un po più ampio e più basso con questo materiale e un 'normal' addizionale nella testura: 
     normal{ ripples 1.15
             scale 0.0125
             turbulence 0.3
             translate<-0.05,0,0>
             rotate<0,-20,0>
           } // end normal

//-----------------------------------------
Materiale per l'acqua con lato superiore ondulato
Adesso tutto che dobbiamo fare è: Aggiungere questo a un bicchiere semplice!
(Il materiale di vetro(glass) a un IOR più alto (~1.5) e un colore un poco differente).
Per più detagli si pùo studiare il testo del file della scena.
