Sky with 2 layers of clouds

For this 3 planes are placed one over the other. Highest level: A plane with sky blue color texture. With "finish{ ambient 1 diffuse 0)" to avoid shadows on the blue of this sky.
Beneath this first plane a second plane with partly transparent cloud structure. This plane forms the high flying clouds. Beneath this plane a third plane with a more transparent cloud structure. These clouds mostly have another direction, which is easy to simulate by scaling this texture.
This is a brighter version of Darin Dugger's great "T_Clouds" from the include file "e;skies.inc"e;. It also is modified to allow the using of "e;fog"e; - which is not possible with the original object "O_T_Cloud2"
