Un ciel avec 2 couches de nuages :

Pour cela on ha besoin de 3 plans placés lun sur l'autre. Le niveau plus 'haut : Un plan avec un ciel bleu, avec "finish{ ambient 1 diffuse 0)" pour éviter des ombres sur le bleu du ciel.
Au-dessous de ce pemière plan un second plan avec une structure de nuages partiellement transparente. Ce plan forme les nuages 'hautes. Sous ce plan un troisième plan avec une structure de nuages plus transparente. Ces nuages ont une autre direction, cela on peut simuler avec des échelles differentes pour x et z.
Cela est une version plus clair les "T_Clouds" de Darin Dugger dans le fichier include "e;skies.inc"e;. Est aussi modifiée pour permettere l'usage de "e;fog"e; - cela n'est pas possible avec l'objet original "O_T_Cloud2"
