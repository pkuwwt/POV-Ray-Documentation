Starry Skies:

For creating an night sky with millions of little stars we can use the starry skies textures made by Jeff Burton called "Starfield1, Starfield2, ..., Starfield6", which we find in the standard include file "stars.inc. We have to use them with an appropriate scale (depends a little bit from your image size!). 

// starry sky ----------------
#include "stars.inc"
sphere{ <0,0,0>, 1
        texture{ Starfield1 scale 0.25
               } // end of texture
        scale 10000
      } //end of sphere ---------------
A starry sky with texture Starfield1.
