Le coucher du soleil :
Pour simuler un coucher de soleil, nous pouvons utiliser un color_map avec un gradient y du bleu foncé au rouge vif sur le sky_sphere. Nous pouvons tourner ce color_map pigment contre l'axe x pour simuler une partie de ciel rougeâtre dans l'ouest (ici: la direction z)
// sky ----------------------------------
sky_sphere{
 pigment{ gradient <0,1,0>
          color_map{
     [0.00 color rgb<0.24,0.32,1> *0.3]
     [0.23 color rgb<0.16,0.32,0.9> *0.9]
     [0.37 color rgb<1,0.1,0> ]
     [0.52 color rgb<1,0.2,0> ]
     [0.70 color rgb<0.36,0.32,1> *0.7 ]
     [0.80 color rgb<0.14,0.32,1> *0.5 ]
     [1.00 color rgb<0.24,0.32,1> *0.3 ]
                   } // end color_map
     scale 2
     rotate <-20,0,0>
     translate <0,0.7,0>
 } // end pigment
} // end sky_sphere ---------------------
Certaines couches colorées de nuages peuvent aider à rendre la scène parfaite.
