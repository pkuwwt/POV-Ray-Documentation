» average «

Une moyenne pondérée pour » piment «, » texture « et » normal «

La Syntaxe de » average « est identique à celle de » color_map «, » pigment_map « etc... Mais : la première valeur derrière les crochets » [ « - correspond à un sens de facteur pour savoir de quelle quantité on l'applique ! En fonction de ces facteurs les piments et les motifs sont une moyenne ! On peut utiliser » average « avec » pigment_map « aussi bien qu'avec » normal_map « ou » texture_map « selon les besoins appropriés.
 
Exemple » average « avec » pigment-map « : 

 
 sphere{<0,0,0>,0.8 scale <1.5,1,1>  translate<0.2,1.1,0>
       texture{
         pigment{ average turbulence 0.75
                  pigment_map{
                   [3 color Yellow] // pondération, pigment
                   [3 Candy_Cane translate<-10,-10,-10> ]
                   [5 color Clear ]
                           } // fin de pigment_map
                  scale 0.5 } // fin de pigment
        finish  { diffuse 0.9 phong 0.5}
              } // fin de texture
      interior{ caustics 0.5 }
      clipped_by{sphere{< 0.5,1,0.5>,1.25}}
    }// fin de sphere ------------------------------
Exemple » average « avec » texture-map « :

 
 sphere{<0,0,0>,1.0 scale <1,0.8,1>  translate<0,1.5,0>
       texture{ average turbulence 0.75
                texture_map{
                 [5 Peel scale 0.25 translate<-30,-30,-30> ]
                 [1 Polished_Chrome
                    normal{bumps 1.5 scale 0.05} ]
                 [3 pigment{ color rgb<1,0.8,0> filter 0.7}
                    finish { diffuse 0.9 phong 0.5}]
                 } // end of texture_map
                scale 1.5} // end of texture
      } // end of sphere -------------------------------------
