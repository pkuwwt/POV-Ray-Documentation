Comment faire

une collection de textures personelles .

Attention, s'il vous plaît! N'ajutes jamais votre textures, couleurs et autres objets personnelles à quelqu'une fichier include standard! Autrement vous pouvez perdre ces objets durant un mise à jour à un version meilleur de POV-Ray.

Vous pouvez ajuter un fichier include nouveau de cette manière :

Pour un fichier include personnelle on écrit les textures dans un fichier comme suit :

 
// Pov-Ray include file Version 3.5 "MesTextures.inc"
// createted by ......
// ---------- textures personelles -----------
#declare Jaune_Chaud = pigment{color rgb<1,0.65,0>}
#declare Eau_Clair  = texture {Glass3
                               normal {ripples 0.5 scale 0.25 }}
// etc. ....
//-------------------------- fin de textures personelles ----

Ce fichier du texte on peut sauvegarder avec l'extension » .inc « dans un registre nouveau et ont peut ajuter à un nouveau » library-path « au fichier du texte Povray.ini
p.ex.: » Library_Path=C:\Programme\POVRAY\INCLUDE\inc « .

Pour utiliser les textures de » MesTextures.inc « on doit inclure ce fichier dans la mème façon que les fichiers include standard » colors.inc « et » textures.inc « :

 
// POV-Ray scene file Version 3.5 "Mon_Test.pov"
// crée par ......
#include "colors.inc"
#include "textures.inc"
#include "MesTextures.inc"
// commencer ici avec la description de la scène
sphere{<0,0,0>,0.5
       texture{pigment{color Jaune_Chaud}
               finish {ambient 0.1 diffuse 0.9 phong 1}
               }// fin de "texture"
      }// fin de "sphere"
// ......
