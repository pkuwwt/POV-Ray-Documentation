  Reguläre Muster  

"brick" = Ziegelstein

Eine recht brauchbare Ziegeltextur mit Standard-Ziegelformat (Maßstab: 1=1m)
ist z.B. folgende Textur: 



texture{
  pigment{ brick
           color White  // Fugenfarbe
           color rgb<0.8,0.25,0.1>// Ziegelfarbe
           brick_size <0.25, 0.0525, 0.125 >
           // Ziegelformat in x-,y-,z-Richtung
           mortar 0.01 // Fugendicke 
         }
  normal { wrinkles 0.75 scale 0.01}
  finish { diffuse 0.9 phong 0.2}
}// end of texture

All diese Muster lassen sich, statt mit Farben("color"), auch mit Pigmenten wie Jade, White_Marble und mit Texturen wie Polished_Chrome und dergleichen verwenden:
z.B.:

 
texture{ pigment { brick
                   pigment{Jade},
                   pigment{White_Marble} 
                 }// end of pigment
         finish  { ambient 0.15 diffuse 0.85 }
        }// end of texture -----------------
