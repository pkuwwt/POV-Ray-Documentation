    Vostre testure
 
                                           
  Motivi regolari:
  "brick" = muro di mattoni.

Una testurea di mattoni in formato standard (scala: 1=1m) molto utile é per esempio la testura seguente: 




texture{pigment{brick color White
                      color rgb<0.8,0.25,0.1>
                       // colore malta, colore mattone
                brick_size <0.25, 0.0525, 0.125 >
                // Ziegelformat in x-,y-,z-Richtung
                mortar 0.01 // spessore di malta
               }
        normal {wrinkles 0.75 scale 0.01}
        finish {ambient 0.15 diffuse 0.85 phong 0.2}
        }// fine di texture

Tutti questi motivi si pùo anche usare con pigmenti e testure come Jade, White_Marble o con testure come Polished_Chrome etc., invece di colori.
Per esempio:

 

texture{ pigment { brick
                   pigment{Jade},
                   pigment{White_Marble}
                 } // end of pigment
         finish  { ambient 0.15 diffuse 0.85 }
        }// end of texture -----------------
