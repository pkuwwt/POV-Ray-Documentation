» pigment_map{...} « - un motivo di pigmenti.



Un esempio per » pigment_map{...} « :

cone{<0,0,0>,1.35,<0,2.00,0>,0
 scale <1,0.6,1> translate<0.3,1.1,0.5>
 texture{
          pigment{ gradient<0,1,0> scale 0.25 turbulence 0.75
                   pigment_map{ [0.00 color Yellow filter 0.25]
                                [0.50 color Red]
                                [0.50 Candy_Cane scale 0.25
                                      translate <-2,-2,0>]
                                [0.80 color Clear ]
                                [1.00 color Clear ]
                              } // end of pigment_map
                 } // end of pigment
          normal { ripples 0.5 scale 0.01}
          finish { diffuse 0.85 phong 1}
         }// end of texture -----------------------------------
  }// end of cone
