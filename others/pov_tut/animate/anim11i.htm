    Oggetti rotanti  

Animazioni cyclici con la rotatione di oggetti.

Rotante oggetto di testo
Un esempio:

// Il testo: ------------------------
text { ttf "arial.ttf", "Rotation", 0.2 , 0
       texture{ pigment{ color rgb<1,0.65,0>}
                normal { bumps 0.5 scale 0.005}
                finish { diffuse 0.8 specular 1}
              } // end of texture
       scale<1,1.8,1>*0.75
       translate<-1.30 ,0.50, -0.0 >
       rotate<0,-360*(clock+0.00),0>
     } // end of text object --------

Descrizione della scena
per POV-Ray:
"rotate00.ini" e
"rotate00.pov"

Una variante più breve: 
text { ...  vedi sopra
       ...
       rotate<0,-180*(clock+0.00),0>
     } // end of text object --------

Oggetti di rotazione con assi di simmetria:
Siamo in grado di applicare la rotazione adattato a la simmetria.
Un cubo: 
 rotate<0,-90*clock,0>
Un hexagon: 
 rotate<0,-60*clock,0>
Una piramide a sette lati con spigoli smussati: 
 rotate<0,-360/7*clock,0>


Descrizione della scena
per POV-Ray:
"rotate10.ini" and
"rotate10.pov".
Per aggiuntivo questa scena ha bisogno del mio include file: shapes_lo.inc!

Una spirale in rotazione:
Basta una semplice rotazione della spirale intorno l'asse x!
