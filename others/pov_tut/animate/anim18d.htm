Psychedelic - Op-Art 

Optische Effekte mit zyklische Animation von Texturen.

Die zyklische Animation von Texturen
Neben der Kamera und geometrischen Objekten lassen sich auch Texturen animieren. Es lassen sich damit Op-Art ähnliche psychodelische, ja geradezu hypnotische Effekte erzielen.
Hier wurde die Phase der "normal"-Komponente einer Textur moduliert: 

normal { onion sine_wave phase -1/4*pi*clock }

Bei diesem Effekt wurde das Muster "spiral1" (ein "pattern") sowohl in einer "color_map" als auch in der "normal"-Komponente angewandt und die Textur dann stetig um die Blickachse gedreht:
