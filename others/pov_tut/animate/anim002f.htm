D'une serie des images
d'animation de POV-Ray
à un fichier de vidéo.

Comment utiliser le logiciel Dave's Targa Animator (DTA) - (gratuit!)
pour convertir une serie des images numérotées
dans un fichier de vidéo avi.
C'est aussi une base pour créer un fichier mpeg
avec des autres logiciels vidéo.
Il n'y a aucune possibilité de faire un fichier avi, mpeg ou gif directement avec POV-Ray.
POV-Ray peut faire seulement une serie des images numérotés nécessaire pour faire un fichier gif ou vidéo.
Ici il y a une méthode très vite et facile pour obtenir un fichier de video : (supposé Windows!)

Étape 0 :   On doit se diriger vers les archives de DTA à
                       ftp://ftp.povray.org/pub/povray/utilities/dta/
                       et télécharger Dave's Targa Animator (DTA) "dta30.zip":

Décompressez le fichier "dta30.zip" dans un registre séparé.
Copiez toutes les images numérotés d'animation de la sortie de POV-Ray (sphere01.bmp, sphere02.bmp, ... sphere30.bmp) dans le même registre.
(Alternativement on peut copier les fichier DAT Windows "dtawin.exe" et "zlib.dll" dans le registre des images numérotés.)

Étape 1:   Démarrez Dave's Targa Animator par double clic sur le fichier "dtawin.exe".
                     Rempliez Fill dans le champ "Script" le nome de base de votres images,
                     mais remplacez le numéro par le joker "*".
Types de fichier d'entrée pour DTA: .bmp, .jpg, .tga
Nouveau dans POV-Ray 3.7: Le type de fichier de sortie par défaut a changé en png (DTA ne comprends pas ".png" !)
Le POV-Ray type de fichier de sortie ".jpg" (nouveau en 3.7) avec une ligne de commande: + fj, est bon avec DTA.
(Il est également conseillé en raison de l'habitude grand nombre de images dans des animations).
Dans POV-Ray 3.6 le fichier de sortie par défaut sur Windows était ".bmp" ! (ligne de commande: + fb nbsp;)


Étape 2: Puis on doit choisir "global options" (le bouton bleu)
                    et sélectionner le type de fichier ("file format") "avi" et cliquer sur buton "OK".


Étape 3: Dans le champ "Output options" on peut voir maintenant "AVI format".
                   Puis cliquer sur le buton de la caméra pour faire le fichier vidéo.
                   On peut voir l'avancement dans le champ au-dessous à la gauche.
                   Et il y a un fichier nouveau dans le registe des images : "anim.avi".



Étape 4: Pour voir le vidéo avec Windows Media Player double cliquez sur fichier "anim.avi".
                   Pour voir des animations cyclique en boucle sans fin on doit cliquer CTRL-T
                   ou activer la boucle sans fin dans les options du Media Player.
