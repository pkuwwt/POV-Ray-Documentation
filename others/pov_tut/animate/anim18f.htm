Psychédélique - Op-Art

Effets optiques avec des animations de textures.
       

Animations cycliques de les textures.
De plus du camera et objets géométrics on peut aussi faire l'animation avec la texture. Nous pouvons obtenir avec cela effets de façon op art avec effects psychodelique et presque hypnotiques.
Qui la phase du composant "normal" d'une texture est modulé : 

normal { onion sine_wave phase -1/4*pi*clock }

Pour cet effet le motif "spiral1" j'avais utilisé deux fois : une fois avec le composant "color_map" et une autre fois avec le composant "normal" component dans une texture tournée continu.
