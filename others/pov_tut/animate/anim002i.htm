Da una serie di immagini di animazione
di POV-Ray a un file video.

Comme usare Dave's Targa Animator (DTA) - (free!)
per convertire una serie di singoli immagini numerati a un file avi video.
Anche comme una base per create un file mpeg con altri programmi video.
Non c'è nessun metodo per produrre un file en avi, mpeg o gif direttamente con POV-Ray.
POV-Ray posso fare solamente una serie d'immagini numerati per convertire in un file gif o video.
Qui una ricetta veloce e facile per ottenere un video file: (presuppongo Windows!)

Passo 0: Si svolge al'archivio di DTA a
                    ftp://ftp.povray.org/pub/povray/utilities/dta/
                    qui si deve download Dave's Targa Animator (DTA) "dta30.zip":

Poi si deve decomprimere il file "dta30.zip" in un directory separato.
Si puo copiare tutte gli immagini numerati di POV-Ray output (sphere01.bmp, sphere02.bmp, ... sphere30.bmp) nello stesso directory.
(Alternativa: si può copiare i file DAT Windows "dtawin.exe" e "zlib.dll" in directory degli immagini numerati)

Passo 1: Si deve iniziare Dave's Targa Animator con doppio clic su file "dtawin.exe".
                   Nel campo "Script" si deve riempire il nome di base degli immagini numerati,
                   intanto si deve sostituire i numeri con la carta bianca "*".
Tipi di file di input per DTA:. bmp, jpg, tga
Novità in POV-Ray 3.7: Il tipo di file di output predefinito è cambiato a png (.! DTA non capisco file png)
Il POV-Ray di tipo file di output ".jpg" (nuovo in 3.7) con una riga di comando: +fj è okay con DTA
. (È anche consigliata a causa del solito grande numero di immagini in animazioni).
In POV-Ray 3.6 il file di output di default in Windows è stato ".bmp"! (riga di comando: +fb nbsp;)

Passo 2: Negli "global options" (il bottone blue) si deve scegliere il formato ("file format") "AVI" e cliccare sul bottone "OK".

Passo 3: Nello spazio "Output options" ora si può vedere "AVI format".
                  Ora si clicca sul bottone "camera" per produrre il file video.
                  Nello spazio a sinistra in basso si può vedere il progresso.
                  E ora si trova anche un nuovo file "anim.avi" nel directory degli immagini numerati.


Passo 4: Per vedere il video con Windows Media Player si doppio clicca sul file "anim.avi".
                  Per vedere una animazione ciclica in un ciclo senza fine si deve cliccare CTRL-T
                  o attivare il ciclo senza nelle opzioni del player.
