Rotierende Objekte  

Über das Rotieren einfacher Objekte

Rotierende Schrift
Ein Beispiel:

// Die rotatierende Schrift: -----------------
text { ttf "arial.ttf", "Rotation", 0.2 , 0
       texture{ pigment{ color rgb<1,0.65,0>}
                normal { bumps 0.5 scale 0.005}
                finish { diffuse 0.8 specular 1}
              } // end of texture
       scale<1,1.8,1>*0.75
       translate<-1.30 ,0.50, -0.0 >
       rotate<0,-360*(clock+0.00),0>
     } // end of text object --------

Szenen Beschreibung
für POV-Ray:
"rotate00.ini" und
"rotate00.pov"

Ein kürzere Variation: 
text { ... siehe oben
       ...
       rotate<0,-180*(clock+0.00),0>
     } // end of text object --------

Rotierende Objekte mit Symmetrieachsen:
Man kann der Symmetrie entsprechend angepasste Rotationen anwenden.
Beim Würfel: 
 rotate<0,-90*clock,0>
Beim Hexagon: 
 rotate<0,-60*clock,0>
Bei der 7eckigen gerundeten Pyramide: 
 rotate<0,-360/7*clock,0>


Szenen Beschreibung
für POV-Ray:
"rotate10.ini" und
"rotate10.pov".
Diese Szene benötigt zudem meine Include-Datei shapes_lo.inc!

Eine rotierende Spirale:
Nur eine einfache Spirale die um die x-Achse rotiert!
