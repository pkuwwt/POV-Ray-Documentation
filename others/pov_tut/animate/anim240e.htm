Realistic Falling and Bouncing
Two macros for simulating falling (slow start, abrupt end) and bouncing (abrupt reflecting): 

//---------------------------------
#macro  Fall_10( X )
  (  sin(pi*(1/2*X + 1/2)) )
#end
//---------------------------------
#macro  Bounce_101( X )
 #if(X0<1/2) (sin(pi*( X + 1/2)))
 #else       (sin(pi*(1-X + 1/2)))
 #end
#end //----------------------------
