Geschlossene Spline-Kurve 
Für die Animationen von zyklisch wiederkehrenden nichtlinearen Bewegungen.

Animationspfade mit Spline-Kurven:
Als Kamerapfad oder als Flugbahn von animierten Objekten läßt; sich u.a. eine geschlossene Spline-Kurve verwenden. 
#declare P1 = <-2.00, 0.20, -2.00>;
#declare P2 = < 1.00, 0.20, -2.00>;
#declare P3 = < 2.00, 0.70, -1.00>;
#declare P4 = < 2.00, 1.20,  2.00>;
#declare P5 = < 0.00, 0.20,  2.00>;
#declare P6 = <-2.00, 3.20,  1.50>;
#declare P7 = <-2.00, 0.70, -1.00>;

#declare Spline_1 =
  spline {
    natural_spline
   -0.250, P7, // Kontrollpunkt
    0.000, P1, // Startpunkt
    0.125, P2,
    0.250, P3,
    0.420, P4,
    0.490, P5,
    0.780, P6,
    0.900, P7,
    1.000, P1, // Endpunkt
    1.125, P2  // Kontrollpunkt
  }// end of spline ---------------
Szenen Beschreibung
für POV-Ray:
"spline00.pov"

Geschlossene Spline-Kurven:
Der Endpunkt ist gleich dem Startpunkt.
Der Kontrollpunkt am Anfang ist der Punkt vor der Endpunkt.
Der Kontrollpunkt am Ende ist der Punkt nach dem Startpunkt.
Die Abstände der Zeitwerte der Kontollpunkte müssen mit denen der entsprechenden Punkte übereinstimmen.
