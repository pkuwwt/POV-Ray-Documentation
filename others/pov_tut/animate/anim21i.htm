Lo Spline chiuso
Per aimazioni di movimenti non-lineari, ciclici e ricorrenti.

Animazione con curve spline: Una curva chiusa spline si può utillizare come sentiero per la fotocamera o per la traiettoria di volo di oggetti animati. 
#declare P1 = <-2.00, 0.20, -2.00>;
#declare P2 = < 1.00, 0.20, -2.00>;
#declare P3 = < 2.00, 0.70, -1.00>;
#declare P4 = < 2.00, 1.20,  2.00>;
#declare P5 = < 0.00, 0.20,  2.00>;
#declare P6 = <-2.00, 3.20,  1.50>;
#declare P7 = <-2.00, 0.70, -1.00>;
#declare P8 = <-2.00, 0.00, -2.00>;

#declare Spline_1 =
  spline {
    natural_spline
   -0.250, P7, // control point
    0.000, P1, // starting point
    0.125, P2,
    0.250, P3,
    0.420, P4,
    0.490, P5,
    0.780, P6,
    0.900, P7,
    1.000, P1, // end point
    1.125, P2  // control point
  }// end of spline ---------------
Descrizione della scena per POV-Ray:
"spline00.pov"

Curva Spline Chiusa:
Si può ottenere una curva spline perfettamente chiuso alle seguenti condizioni:
Il punto finale è lo stesso come il punto di partenza.
Il punto di controllo prima che il punto al inizio è il punto prima che il punto di fine.
Il punto di controllo alla fine è il punto dietro al punto di partenza.
Le distanze dei valori del tempo al punto di controllo deve anche adatto a quelle dei loro punti base.
