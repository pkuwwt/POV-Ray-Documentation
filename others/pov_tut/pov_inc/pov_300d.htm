Der Ausgabeformatbefehl für das gerenderte Bild


Output File Type - Das Ausgabedateiformat der gerenderten Bilder
In der Grundeinstellung ist der Ausgabedatei-Typ für die gerenderten Bilder in Windows
  mit POV-Ray 3.7 : portable network graphics, Datei-Erweiterung: '.png'
  mit POV-Ray 3.6.2: Windows bit map format, Datei-Erweiterung: '.bmp'

Andere Ausgabedateitypen:
 Format  Erweiterung  Kommando-Zeilen
 Shortcut		  Ini file Kommando
 Langversion
 Jpeg format *)  .jpg  +fj  Output_File_Type=J
 Targa, uncompressed Targa-24 format  .tga  +ft  Output_File_Type=T
 png, portable network graphics  .png  +fn  Output_File_Type=N
 ppm, Unix PPM format  .ppm  +fp  Output_File_Type=P
 bmp, Windows bit map format  .bmp  +fb  Output_File_Type=B
 HDRI, OpenEXR, High Dynamic-Range format *)  .exr  +fe  Output_File_Type=E
 HDRI , Radiance high dynamic-range format *)  .hdr  +fh  Output_File_Type=H
*)Anmerkung: Ausgabe in Jpeg und HDRI format, ist nun verfügbar mit POV-Ray 3.7 (nicht mit 3.6.2)!
  Weitere Information über Jpeg-Kompressionsfaktoren, HDRI-Bilder, bits per pixel und alpha transparency data
  siehe in der (engl.)POV-Ray-Hilfe (Suche nach 'output')

POV-Ray liest die Hilfsdateien 'povray.ini' und 'quickres.ini' nur einmal während dem Programmstart.
Jegliche Änderung an diesen Dateien benötigt ein Schließen und einen Neustart
des POV-Ray-Editor-Programms um wirksam zu werden!
