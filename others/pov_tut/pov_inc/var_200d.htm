Mathematische Ausdrücke
...

In der Szenen Beschreibungssprache (SDL) von POV-Ray kann man ein Bündel von
mathematischen und logischen Operatoren für Berechnungen in POV-Ray verwenden.
Operatoren in POV-Ray:

Mathe Operatoren
(  )  *  /  +  -    und logisches NICHT:  ! 
#declare Result = -0.45*A + (B*B - 4)*C; //
#local Angle_1 = degrees( atan ( 2.5 / sqrt(2)); // Winkelwert in Grad


Relationale Operatoren.
<  <=  =  !=  >=  > 
 #declare Green_or_Red = pigment{ rgb<(clock <= 0.5),(clock > 0.5),0;>,}


Logische Operatoren
  &  // = UND
  |  // = ODER 
  &! // = UND NICHT
//Anmerkung: Die Argumente A und B werden nicht bitweise sondern logisch behandelt! 
#if (A & B)
  ... // mach dies, wenn beide A und B wahr sind
#end
#if (A | B)
  ... // mach dies, wenn entweder A oder B wahr ist
#end
Achtung: Jedes #declare oder #local von Zahlen, Vektoren oder Farben (color) verlangt ein Semicolon am Ende!
