Eigene Elemente zum Insert-Menu hinzufügen.
(POV-Ray for Windows version 3.6 oder 3.7)

Das Objekt, das man zum 'Insert Menu' hinzufügen will, sollte in handlich kleiner 'Thumbnail'-Auflösung wie etwa 160 x 120 Pixel in POV-Ray gerendert werden. Dabei sollte eine Ansicht gewählt werden, die alle essenziellen Details des Objekts klar darstellen. Sodann macht man eine kleine Textdatei, die alles beinhaltet, was zum Einfügen in eine POV-Ray-Szenendatei diese Objektes notwendig ist:
Thumbnail-Bild von '10 - Table_01.jpg'
für das 'insert menu'.
 
//------------------------------------
#include "Table_01.inc"
//------------------------------------
object{ Table_01
        scale<1,1,1>
        rotate<0,0,0>
        translate<0,0,0>
      }
//------------------------------------
Die Textdatei '10 - Table_01.txt'
für das 'insert menu'

Nun erzeugt man ein Unterverzeichnis für unsere eigenen Insert-Menu-Sachen.
Man bezeichnet es z.B. mit '00 - MyInsertMenu'. (Der Teil '00 - ' wird im POV-Ray-Editor nicht zu sehen, die Nummern bestimmen lediglich die Reihenfolge in der die einzelnen Elemente sortiert erscheinen werden!)


Man muß nun dieses Unterverzeichnis öffnen und die beiden Dateien (das Bild und den kurzen Szenentextabschnitt) mit demselben Namen und derselben Nummerierung dort einfügen. (In POV-Ray 3.62 muß das Thumbnail-Bildchen im Format '.bmp' sein!)

Wenn wir nun den POV-Ray-Editor mit einen Szenentext öffnen, können wir unser neues Objekt wie all die übrigen Objekte über das Insert-Menü einfügen:
Verwendung des Objekts 'Table_01' in einer Szenendatei über das 'insert menu'.
