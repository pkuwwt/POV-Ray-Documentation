Schleifen und verdrehte Spiralen

Im den letzten Beispielen hatten wir meist Kugeln verwendet. Wenn man statt dessen Ellipsoide verwendet, besteht noch die Möglichkeit eine Drehung wie in den Möbius-Schleifen anzuwenden.

Zunächst eine Spirale mit Ellipsoiden: 

//------------------------------------
#declare Ball =
 sphere{<0,0,0>,0.45
        scale <0.7,1.5,1> // !!!
        texture{
         pigment{color rgb<1,0.6,0>}
         finish{ phong 1}}}

//der Rest wie im vorherigen Beispiel!

Szenenbeschreibung für POV-Ray: povloop9b.pov
oder povloop9b.txt

Das verwendete Ellipsoid nun zusätzlich kontinuierlich um die z-Achse gedreht: 

//------------------------------------
...
#while (Nr< EndNr)
 object{Ball
        rotate <0,0,Nr*0.5>  // !!!
        translate<Radius0+Nr*D,0,0>
        rotate<0,Nr * 360/N_p_rev ,0>}
 #declare Nr = Nr+1;  // next Nr
...
//------------------------------------
