x = <1, 0, 0>; y = <0, 1, 0>; z = <0, 0, 1>;
xy = < 1, 1, 0>; yz = < 0, 1, 1>; xz = < 1, 0, 1>;
t = <0, 0, 0, 1>; u = <1, 0>; v = <0, 1>;

<a,b,c> + <d,e,f> = <a+d,b+e,c+f>

Extrahieren von Komponenten
