   Anche update del mioInsert Menu Add-on per POV-Ray 3.7 (20-Nov-2014)
A causa delle innovazioni essenziali si è raccomandato urgentemente di usare POV-Ray 3.7:
La capacità di SMP (Symmetric multiprocessing) con i processori multicore può accelerare il processo di rendering molto ( per es. Intel i7 -> 4 cores -> 8 threads parallelo -> ca. 8 più veloce! ).
E in più POV-Ray 3.7 ha un 'Insert Menu' rivisto intensamente. Il contiene oggi i parti più importanti de mio 'Insert Menu Add-on' per POV-Ray 3.6.2.
Nouovo è anche il for loop (#for(COUNTER,START,END,STEP) .... #end), diverse nuove funzioni di motivi e la possibilità de creare gli immagini non soltanto in formato JPEG (poco bisogno di spazio!) ma anche in formato .hdr.
Inoltre si può anche usare immagini HDRI per illuminare indirettamente la scena (HDRI environment mapping).
Il mio precedente file di inclusione 'shapes_lo.inc' è sostituito con POV-Ray 3.7.0 con il file migliorato di 'shapes3.inc'. Questo file e il testo associato dei modelli di Insert Menu può già essere scaricato sul mio pagina di download .
Buon divertimento con POV-Ray !
Friedrich A. Lohmüller
04 genaio 2014

   Aussi mise-à-jour mon Insert Menu Add-on pour POV-Ray 3.7( 20 Novembre 2014 ).
À cause des innovations techniques essentielles il est très recommandé de utiliser POV-Ray 3.7 :
La capacité de SMP (Symmetric multiprocessing) avec les processeurs multicore utilisés aujourd'hui peut fortement accélérer le processus de rendering (par ex. Intel i7 -> 4 cores -> 8 threads parallèle -> ca. 8 fois plus vite ! ).
Outre cela POV-Ray 3.7 a un 'Insert Menu' retouché très intensive. Il contient maintenant les parties plus intéressant de mon 'Insert Menu Add-on' pour POV-Ray 3.6.2 existé.
Nouveau est entre autres aussi la boucle for (#for(COUNTER,START,END,STEP) .... #end), divers nouveau fonctions de motif et la posibilité de creer les images non seulement en format JPEG (très peu encombrant !) mais aussi en fromate .hdr. De plus on peut maintenant illuminer les scènes indirectement par images HDRI ( HDRI environment mapping ).
Mon précédent fichier include 'shapes_lo.inc' est remplacé avec POV-Ray 3.7.0 par le fichier amélioré de 'shapes3.inc'. Ce fichier et le texte associé des modèles de Insert Menu déjà peuvent être téléchargés sur mon page de téléchargement.
Je vous souhaite un bon tracé des rayons avec POV-Ray.
