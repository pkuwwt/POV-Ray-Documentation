                                        		Comment créer une Scène en POV-Ray

Créer une scène avec POV-Ray pour Windows est très facile à faire avec un effort minimal d'écrire.
Pour cela nous utilisons les modèles de texte de l'» Insert Menu « POV-Ray 3.7.
POV-Ray 3.6.2 : Avec POV-Ray 3.6.2. il est très recommandé utiliser ma extension » Insert Menu Add-on «. Avec cet additif la creation des scènes avec POV-Ray est devenu plus convivial et plus vite !
Il y a aussi une version HTML pour l'usage avec copier-coller ('cut and paste') anche pour des systèmes non-windows - dans 'C:/Users/Documents/POV-Ray/v3.7/scenes/templates/_index.htm'.
    Vous pouvez avoir mon » Insert Menu Add-on « (aussi la versione HTML pour téléchargement et comme version en ligne) et autres additifs à mes pages de descriptions et exemples à notre page d'accueil http://www.f-lohmueller.de/pov_tut/pov__fra.htm - voyez ici: »Téléchargement«.

Pour obtenir une première vision de l'apparence de notre scène vide nous choisissons une résolution convenable avec le menu drop-down au-dessus du bouton » New «.
Puis nous sauvegardons notre texte de scène avec le bouton » Save « - sil vous plaît n'utilisez pas aucune des caractères non-ASCII en évitant les accents en français, aucune de barres d'espacement et aucun caractères spéciaux à l'exception de »_« dans votre nome de fichier (dans l'exemple ici: » demo1 «). L'extension » .pov « est ajoutée automatiquement.
Cliquer sur » Run « présente la scène dans un » window « additionnel, nommé le » Render Window «:

V.    Comment éditer notre Fichier de Scène?

Pour modifier le texte de scène nous avons plus des outils que seulement insérer quelques modèles de texte de l'» Insert Menu «. Nous pouvons faire tous les manières des traitement de texte que nous savons de les éditeurs de texte modernes comme p.ex. "Word".
Quelques des les fonctions plus importantes:
Ctrl + c copie une section de texte macquée dans le mémoire vive.
Avec Ctrl + v cette section de texte est insérée au point indiqué par le curseur.
La même chose on peut faire avec Ctrl + Insert.

Ce piège vous trouveront dans tous les éditeurs de texte:
» Mon curseur mange les caractères! « - (au lieu de insèrer les caractères!)
Normalement (cela est indiqué par le signe » Ins « dans le quatrième secteur gris en base) un éditeur de texte insère un caractère au point indiqué par le curseur. Mais se nous cliquons sur la touche »Insert« une fois l'éditeur change au mode pour écrire et effacer (» overwrite «) (cela est indiqué par le signe » Ovr « dans le quatrième secteur gris en base).
Comment changer au retour: On doit seulement cliquer encore une fois la touche »Insert« !

La function très important "undo" (arrière!) est comme dhabitude;Ctrl z: Avec cela est possible de annuler toutes les pas antérieures.
La commande correspondante est Ctrl y (): redo ("répéter")
Cettes commandes on trove aussi avec un "clic de la souris avec touche droite" ou avec le menu en 'haut "Edit"!


VI.    Où est l'Image?

Se vous ne savez pas où se trouve l'image rendu - normalement dans le même registre comme le texte du scene (.pov). Pour changer le chemin : Cliquez dans votre l'éditeur de POV-Ray sur » Tools «, » Edit master POV-Ray.INI «:
Vous ajoutez quelque chose comme cela au fin de ce fichier de texte:
» Output_File_Name=C:\User\POV-Ray_images\ «.
Vous pouvez changer cette ligne à chaque chemin désiré. Se votre fichier de texte est nommé p.ex. » Mon_test1.pov « vous trouvez dans ce chemin und fichier de image bmp nommé » Mon_test1.bmp «
Mais n'oubliez pas le backslash au fin du texte, autrement tous les images rendues sont sauvegardées avec le nom » images.bmp « - et toutes les images plus vieilles sont effacées.
