Rail Track System für POV-Ray
Y-Weichen und 3Wege-Weichen

Y-Weiche 
//--------------------------------------------
#include "RT_System/RT_Track_Switch_Y_00.inc"
// returns: 'Switch_Length_', 'Rail_Distance_' !
//--------------------------------------------
object{ RT_Track_Switch_Y_00(
         1, //direction: 0= right,1=left
         90.80, //Switch_Radius,
         15.00, // Switch_Angle,
         1.5 // Ties_Per_Meter  // ~1.5
        scale<1,1,1> rotate<0,0,0>
        translate<0.00,0.00,0.00>
      } //------------------------------------

RT_Tracks_Switch_3Ways_B_00
Geometrie Typ B.
Anmerkung: Man kann Weichen aus derselben Include-Datei mit verschiedenen Parameter nacheinander verwenden, aber man sollte dann
POV-Ray dazu zwingen die entsprechende Include Datei erneut zu laden bevor man die Weiche mit geänderten Parametern aufruft:
Z.B.: Das Makro RT_Track_Switch_3Ways_A_00(...) deklariert selbst die Werte: 'Switch_Radius_', 'Rail_Distance_'. 
#undef RT_Track_Switch_3Ways_A_00_Inc_Temp
#include "RT_System/RT_Track_Switch_3Ways_A_00.inc"
Die garantiert, dass die bereits berechneten und deklarierten Werte von der Include-Datei korrekt neu berechnet werden.
