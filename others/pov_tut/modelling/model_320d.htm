Rail Track System für POV-Ray
Geraden & Kurven

Gerade Gleise 
//---------------------- straight track in x+
#include "RT_System/RT_Track_Straight_00.inc"
//--------------------------------------------
object{RT_Track_Straight_00(
           15.00,// Track_Length,  // meter
           1.5, // Ties_Per_Meter // ~1.5,
         ) //---------------------------------
        rotate<0,0,0>
        translate<0.00,0.00,0.00>
      } //------------------------------------
Die Schwellen werden äquidistant verteilt!
RT_Track_Straight_00

Kurven Gleise 

//-------------------- right curve track in x+
object{ RT_Track_Curve_00(

#include "RT_System/RT_Truck_Curve_00.inc"
//---------------------------------
           30.00, // Rail_Radius,   // meter
           30,    // Bow_Angle,     // degrees
           1.5,   // Ties_Per_Meter // ~1.5,
         ) //---------------------------------
        rotate<0,0,0>
        translate<0.00,0.00,0.00>
      } //------------------------------------
Die Schwellen werden äquidistant verteilt!
Rotation: mittels   #include "transforms.inc" kann man z.B.
Rotate_Around_Trans(<0,Angle,0>,<0,-Rail_Radius,0>) verwenden.
Achtung:
      Der maximale Winkel dieses Kreisbogens beträgt 90° !!!!
