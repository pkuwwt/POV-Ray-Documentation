 Usage simplifié du
  of my 'Rail Track System pour POV-Ray'
  avec le fichier include 'RT_System_00.inc'

Les qualités: Ce fichier include combine tous les fichiers include des éléments de base de voies ferrées
de mon 'Rail Track System pour POV-Ray'. Échelle : 1 unité = 1 m (metre).
Il met aussi à la disposition diverses types de simulation.

Types of Simulation :
  0 = voies avec traverses (ties),
        au niveau zéro.
  1 = lamelles doubles sans traverses,
        texture: 'Tracks_Lines_Texture', au niveau zéro.
  2 = blocs simples (h = 0.20),
        texture: 'Tracks_Quick_Texture'
  3 = blocs avec marques aux bouts,
        texture: 'Tracks_Quick_Texture'
 -1 = voies avec une base de 'box',
        texture: 'Tracks_Quick_Texture'
 -2 = voies avec ballaste,
        texture: 'Gravel_Texture'
    ( hauteur du base / ballast = 0.20,
    Soulevez les voies par <0,0.20,0> sur zéro ! )

Changement des textures
Ou nous devons déclarer simplement la texture avant
inclure le fichier 'RT_System_00.inc' ou nous devon
'undef'inir la variable 'Gravel_Texture' déclaré avant comme cela : 
#undef Gravel_Texture
#declare Gravel_Texture =
   texture{
     pigment{ color rgb <0.79,0.60,0.52> }
     normal { bumps 5 scale 0.035 }
     finish { phong 0.5 } } // gray brown
