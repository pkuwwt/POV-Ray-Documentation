Why this?   I have made this Insert Menu Add-on for POV-Ray to make creating scenes easier
for beginners and also more comfortable for advanced users.
There are templates for basic scenes, completely textured shapes with vanilla transformations, etc..

How to use?
Windows Version 'Insert Menu Add-on' for POV-Ray:
   After having installed this add-on, all you have to do for building your own new scene is:
Step 0: Click on "New" button and you get a new white sheet for a scene text. Click on "Save" choose a good name (without anything than normal ASCII characters!) and save this empty text file like you save any normal text file but ending with ".pov" (The editor used with POV-Ray for Windows will do this automatically!).
Step 1: Click on "Insert Menu", "Basic templates", "Ready_made scenes" and choose one of the scenery backgrounds (Do this only one time in a scene!).
Step 2: You move your cursor at the end of the text! (Don't forget this - otherwise you ... will see not very much!) Click on "Insert Menu", "Basic templates" "Shapes with textures" and choose the shape you want to add in your scene.
Step 3: Click on "Run" button and you will see this scene rendered.
All shapes have colors and other surface properties in textures statements. All shapes also have vanilla transformations like scale<1,1,1>, rotate<0,0,0>, translate<0,0,0>. This makes it very easy, to play around with them and gain experience, because you can't make so much errors in spelling and punctuation.

HTML-Version / Online-Version for all POV-Ray versions:
   Unzip the compressed file of the HTML-Version and open the file '_index.htm'
with any webbrowser parallel to your POV-Ray Scene Editor.
No automatic insert available - we have to insert the templates by 'Cut and Paste'.
Klick on the thumbnail image of the element you want to insert. A new browser window with the according template text file will open. Pick from this browser window (mark it and hit CTRL-C) the whole text from the browser window by 'Cut (CTRL-C) and Paste(CTRL-V)' and paste it with CTRL-V in your scene file where you want it to be inserted.
