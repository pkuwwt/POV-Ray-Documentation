Insert Menu Add-on (POV-Ray versione 3.6.2 / 3.7)
Aggiornamento dei modellí di testo Insert Menu Add-on: 24 dicembre 2014
Aggiornamento di shapes3.inc: 08 ottobre 2014
Mac: PovRay 3.7 Unofficial for Mac now fully supports the standard insert menu from the Windows version, including the preview images. There is even an option to scale the images down for lower screen resolutions, the insert menu doesn’t have to be in a particular place because you can select the folder in the preferences. See here: http://megapov.inetart.net/povrayunofficial_mac/. (16 novembre 2013)
Linux: Ho anche rapporti che il funziona cón POV-Ray for Linux e il neXtgen Povray Editor (ma solo senza gli immagini thumbnail!). Vede qui: neXtgen Povray Editor. [Un editor più vecchio: QTPov-Editor Vede qui: http://qtpoveditor.sourceforge.net/] (21-giugno-2009). Alternativamente è possibile di usare 'POV-Ray for Windows' con 'Insert Menu Add-on' e con suo editor eccellente, con qualche restrizione piccola, anche sotto Wine (25 aprile 2010))

Perchè questo?
Ho fatto quest'aggiunta "Insert Menu Add-on per POV-Ray versione Windows" per facilitare creare scene per i principianti ma anche per fare l'uso più comodo per gli utenti advanzati.

Come usare?
Versione 'Insert Menu Add-on' per POV-Ray for Windows:
   Dopo l'installazione di quest'Add-on tutto che abbiamo a fare per creare una scena nuova è:
Passo 0: Clicciamo su "New" e abbiamo un foglio in bianco per il testo nuovo. Poì clicciamo su "Save" e scegliamo un nome buono (senza qualcos'altro che caratteri di ASCII normali!) e salviamo questo testo vuoto come un testo normale di text file con desinenza ".pov" (L'editor di POV-Ray per Windows3.5. farà questo automaticamente!).
Passo 1: Clicciamo su "Insert Menu", "Basic templates", "Ready_made scenes" e scegliamo uno dei sfondi di scena della sezione "Ready_Made scenes" (Faccia questo solo una volta per ogni scena!).
Passo 2: Moviamo il nostro cursore al fine del testo! (Non dimenticare questo - in caso contrario ... non vedrò molto!) Clicciamo so "Insert Menu", "Basic templates" "Shapes with textures" e scegliamo una figura geometica che vogliamo aggiungere alla nostra scena.
Passo 3: Clicciamo su "Run" per vedere questa scena renderizzata.
Tutte le figure geometiche sono colorate completamente e hanno di pseudotrasformazioni per facilitare giocare con loro.

Versione HTML / Versione Online per tutte le versioni di POV-Ray:
   Unzip il file compresso della versione HTML e apri il file '_index.htm'
con un webbrowser in parallelo al editore di POV-Ray.
Un inserimmento automatico non è disponibile - si deve inserire i modelli con 'Cut and Paste'.
Chicciamo sul imagine thumbnail del'elemento que vogliamo inserire. Una finestra di browser nuova si apri con il testo del modello. Copiare con 'Cut (CTRL-C) and Paste(CTRL-V)': Si può marcare e copiare il testo in questa finestra (marcare e preme il tasto CTRL-C) e inserire il testo con CTRL-V nel file di scena dove vogliamo inserire il modello.

Versione HTML del'Insert Menu Add-on:
