Insert Menu Add-on (POV-Ray version 3.6.2 et 3.7)
Mise-à-jour des modèles de texte Insert Menu Add-on : 24 décembre 2014
Mise-à-jour de shapes3.inc : 08 octobre 2014
Mac : PovRay 3.7 Unofficial for Mac maintenant peut fonctionner avec l'insert menu de la version pour Windows, aussi avec les images de aperçund. Il y a aussi une option de scaler les images plus petit pour des resolutions de l'écran plus faible. L'insert menu ne doit pas être à un lieu specifique, on peut selectione le lieu dans 'preferences'. Plus ici: http://megapov.inetart.net/povrayunofficial_mac/. (16 Novembre 2013)
Linux : On m'en aussi a donné que il peut fonctionner, avec POV-Ray for Linux et l'éditeur neXtgen Povray Editor (excepté
les images de aperçund) Voyez ici : neXtgen Povray Editor. [ Un editeur plus vieux : QTPov-Editor Voyez ici :
http://qtpoveditor.sourceforge.net/ ] (21 juin 2009). Alternativement il est possible d'utiliser 'POV-Ray for Windows' avec
'Insert Menu Add-on' et avec son éditeur excellent, à quelques petites réserves, aussi sous Wine (25 avril 2010))

Pourquoi ce menu?
J'ai fait ce menu intégré pour la version POV-Ray windows pour faciliter la tâche des débutants and rendre l'environnement plus convivial pour les utilisateurs avancés.

Comment l'utiliser?
Windows Version du 'Insert Menu Add-on' pour POV-Ray:
    Après avoir installé le menu, tout ce qui vous reste à faire pour construire votre scène est le suivant :
Etape0 : Cliquez sur l'option « Nouveau » fichier, apparaît une feuille de texte scène. Cliquez sur « sauve » et choisissez un nom ressemblant au modèle que vous être en train de faire (surtout n'employez pas d'autres lettres que celle du standart ASCII ! ) et sauvez ce fichier texte vide, comme vous le feriez avec un fichier ordinaire, mais l'extension avec « .pov » ( L'éditeur de POV-Ray vas le faire automatiquement ! ).
Etape1 : Cliquez sur "Insert Menu", "Basic templates", "Ready_made scenes" et choisissez un des arrières plans scèniques (Ne faites cela qu'une fois par scène, sinon vous en aurez autant que de fois que vous cliquez ! (Si vous faites une erreur cliquez sur Edition annuler ! ).
Etape2 : Déplacez votre curseur à la fin du text ! (Surtout n'oubliez pas de le faire ! Sinon vous ne verriez pas grand chose ! ) Cliquez sur "Insert Menu", "Basic templates" "Shapes with textures" et choisissez la forme que vous voulez ajouter à votre scène.
Etape3 : Cliquez sur l'option "Run" (exécuter) et vous verrez le rendu de scène.
Toutes les formes ont une « texture » et possèdent également des transformations « vanilla » (bidons) afin de pouvoir jouer avec.

Version HTML / Version en ligne pour toutes les versions de OS:
   On doit décompresser le file de la version HTML et ouvrir le ficher '_index.htm'
avec un navigateur parallel au éditeur de POV-Ray.
L'insertion automatique n'est pas disponible - nous devon insérer les modèles par 'Cut and Paste'.
On doit sur un image thumbnail de l'élément desiré. Une nouveau fenêtre s'ouvre avec le text modèle correspondant. On peut picoter ce texte de la fenêtre du navigateur (marquer et presser CTRL-C) par 'Cut (CTRL-C) and Paste(CTRL-V)' et on peut insérer le text par CTRL-V dans notre fichier de la scène de POV-Ray.

Comment installer l''Insert Menu Add-on' avec POV-Ray for Windows :
Attention , sil vous plaît! : Si vous avez déjà installée une version de "Basic templates" précédente, il est indiqué de effacer ou débaptiser cette version. Autrement vous avez une mélange chaotique de modèles de texte vieux et nouveau!

Étape 1 : Téléchargez mon » Insert Menu Add-on « avec le fichier
Basic_templates_37.zip (7.961 kB) 24 décembre 2014, pour POV-Ray 3.7! (images .jpg).
Basic_templates_36.zip (18.830 kB) 06 avril 2013 (images .bmp).
et copiez cet fichier à votre chemin pour l'» Insert Menu « dans le chemin de POV-Ray.
(Normalement c'est avec POV-Ray 3.7:
 sous Windows XP/Vista/Windows 7: » ..\Documents\POV-Ray\v3.6 bzw. v3.7\insert Menu\«.
 Ou avec POV-Ray for Windows 3.62 :
 Windows XP : » C:\Programmes\POV-Ray for Windows v3.6\insert Menu\«
 Windows Vista : » C:\Utilisateurs\MON_NOM\Documents\POV-Ray\v3.6\insert Menu\« )
