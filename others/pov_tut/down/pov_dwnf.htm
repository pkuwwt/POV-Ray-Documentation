.  		 Pour commencer plus facil je fait un paquet des modèles de texte pour moi
» Insert Menu Add-on « à POV-Ray version 3.6.2/3.7 pour Windows un additif à les modèles de texte original del' Insert Menu et aussi une version en HTML pour tous les OS.
Mise-à-jour : 24 décembre 2014.
Ces modèles de texte sont fonctional sans ajouter aucun nombres et fourniron quelques scènes de base. C'est utile pour mieux faire comprendre la fonction de les fichiers de scène en POV-Ray.
Fichiers include : shapes_lo2.zip,analytical_g.inc (shapes_lo.inc est maintainant dans POV-Ray 3.7.0 comme shapes3.inc ).
