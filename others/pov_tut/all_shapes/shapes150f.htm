box = cuboïde carré, boite carrée,
parallélépipède rectangle
syntaxe générale : 
box{ <x1,y1,z1>,<x2,y2,z2>
     texture{ ... ... }
     rotate<0,0,0>
     translate<0,0,0>
   } ---------------------
Ici "<x1,y1,z1>" et "<x2,y2,z2>" sont sont les coordonnées de deux points diamétriquement opposés.
Exemple : 
box{ <2,0,1>, <5,2,6>
     texture{ ... ... }
   } // -----------------
