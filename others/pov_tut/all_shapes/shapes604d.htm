Nationalen Sonderzeichen mit Unicode

Die installierten Schriftarten ("fonts") erhalten in der Regel wesentlich mehr Zeichen als den normalen ASCII-Zeichensatz. Verwendet werden können alle auf dem jeweiligen System installierten Schriftarten. Probleme bereiten jedoch die deutsche Umlaute sowie alle nationale Sonderzeichen, da POV-Ray in der Grundeinstellung nur die reinen ASCII-Zeichen beherrscht.
Um dann die in den jeweiligen Schriftarten enthaltenen Sonderzeichen in text-Objekten verwenden zu können benötigt man in POV-Ray vorweg das Kommando:
global_settings { charset utf8 }


Unicode-Text-Objekt
mit Umlauten und Sonderzeichen

Unicode mit dezimalen Buchstabennummern: 
global_settings { charset utf8 }
text{
  ttf "arial.ttf",
  concat(
  chr(196),chr(203),chr(214),chr(220),
  " ",chr(8364)), 0.01, 0
  pigment{color rgb<0.5,0,0>}
  scale 0.4 translate<-0.5,0.5,0> }

Alternativ mit hexadezimalen Buchstabennummern: 
global_settings { charset utf8 }
text{
  ttf "arial.ttf",
  "\u00C4\u00CB\u00D6\u00DC \u20AC",
  0.01, 0
  pigment{color rgb<0.5,0,0>}
  scale 0.4 translate<-0.5,0.5,0> }
