                                        		       Supercone macro
          En quelques situations la version originale déclaré dans "shapes.inc" ne marche pas correctement.
Solution : #include "SuperCone.inc" - Ce fichier include avec une version corrigée du macro "SuperCone" par Warp 20.10.2001 - doit être intégrée après le fichier "shapes.inc" . 
// ------------------------------------
object{ Supercone(
        // point A, axis Ax, axis Az,
        <0.0,0.0,0.0>, 0.5, 1.0,
        // point B, axis Bx, axis Bz)
        <0.0,1.5,0.0>, 0.7, 0.4)

        texture{
          pigment{ color rgb<0.65,1,0>}
          finish { phong 1}
        } // end of texture

        scale<1,1,1>
        rotate<0,0,0>
        translate<0,0,0>
      } // ----------------------------
