          Die originale Version deklariert in "shapes.inc" ist in bestimmten Situationen fehlerhaft.
Problemlösung: #include "SuperCone.inc" - Diese Include Datei mit der korregierte Version des Makros "SuperCone" von Warp 20.10.2001 - sollte nach der Datei "shapes.inc" eingebunden werden. 
// ------------------------------------
object{ Supercone(
        // point A, axis Ax, axis Az,
        <0.0,0.0,0.0>, 0.5, 1.0,
        // point B, axis Bx, axis Bz)
        <0.0,1.5,0.0>, 0.7, 0.4)

        texture{
          pigment{ color rgb<0.65,1,0>}
          finish { phong 1}
        } // end of texture

        scale<1,1,1>
        rotate<0,0,0>
        translate<0,0,0>
      } // ----------------------------
