Paraboloides et Hyperboloides du fichier include "shapes.inc"

Avant de pouvoir utiliser les formes géométriques le fichier include doit être inséré dans la description de scène par :
#include "shapes.inc"

 

Paraboloides :
(en direction X, Y, Z positive)		 Hyperboloides :
 
Paraboloid_X,
  		Hyperboloid     (la selle à gauche)
Paraboloid_Y Hyperboloid_Y
 
Paraboloid_Z 		(double cone dans la direction y, voir dans le milieu, tourné vers la droite par "rotate<90,0,0>")

Ce sont des surfaces infinies qui peuvent être utilisées dans les fonctions csg, telle que "intersection" ou "clipped_by" avec des formes finies pour obtenir celles-ci de cette façon. Ces formes géométriques sont définies en tant que symetric à <0,0,0>. Exemple pour montrer comment les utiliser comme objet : 
object{ Paraboloid_Y
        texture{....}
        translate<...> }
