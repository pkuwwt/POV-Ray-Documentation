                                        		 Round_Cone - cône arrondi
    Declaré en "shapes.inc"
Round_Cone
type 1 
// ---------------------------
object{
 Round_Cone(
  //point A, radius A:
  <0,0,0>,0.70,
  // point B, radius B:
  <0,1.2,0>, 0.40,
  0.15, // rounding radius,
  0)// merge on

 pigment{
 color rgb<0.75,0.35,1>}

 scale<1,1,1>
 rotate<0,0,0>
 translate<0,0.0,0>
} //--------------------------
 Round_Cone2
type 2 
// ---------------------------
object{

  //point A, radius A:
  <0,0,0>,0.70,
  // point B, radius B:
  <0,1,0>, 0.40,
  // merge on
  0)//

 pigment{
 color rgb<0.75,0.35,1>}

 scale<1,1,1>
 rotate<0,0,0>
 translate<0,0.0,0>
} //--------------------------
 Round_Cone3
type 3 
// ---------------------------
object{
 Round_Cone3(
  //point A, radius A:
  <0,0,0>,0.70,
  // point B, radius B:
  <0,0.8,0>, 0.40,
  // merge on

 pigment{

 scale<1,1,1>

} //--------------------------
Note :
Toutes ces " round cones " sont declarés en macros par unions.
Round_Cone est une union d'un cône et deux two tores.
Round_Cone2 et Round_Cone3 sont des unions d'un cône et deux sphères.
Pour cette raison, quant nous prenons des textures ou materials transparents,
tous les trois ont besoin d'être declarés comme " merge " par activer le merge flag " 1 " !

Cettes trois formes sont declarées dans le fichier include "shapes.inc".
