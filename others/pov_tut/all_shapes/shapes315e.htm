                                    		  Disk_X, Disk_Y, Disk_Z - Cylinders in one unit size in the directions of x, y , z.
  Declared in "shapes.inc"
Disk_X 
// --------------------------
object{
 Disk_X // no open available!
 scale <1,1,1>
 texture{
  pigment{color rgb<.8,.5,1>}
  normal {bumps .5 scale .025}
  finish {phong 1 }
  } // end of texture
  rotate<0,0,0>
  translate<0,0,0>
} // end of disc -----------
