Questo oggetto è definito con una macro nel mio include file "shapes3.inc" .
 Round_Conic_Prism
Sintassi generale: 
object{ Round_Conic_Prism(
                         Center_Distance,
                         R_upper,
                         R_lower,
                         Length, // in z-
                         Border_R,
                         Merge_On
                       ) //--------------
         texture{ ... }
      } // end of object ----------------
Center_Distance =
      distanza verticale dei centri di tori,
R_upper >0, raggio a <0,C_distance,0>,
R_lower >0, aggio a zero.
Length >2*Border_R, lunghezza di prisma in z-
Border_R = raggio del bordo, max.= min(R_lower,R_upper)
Merge_On: 0 = union, 1 = merge.
Archivio esempio
".txt" o ".pov"
        Esempio: 
#include "shapes3.inc"
object{ Round_Conic_Torus(
           0.95, // >0, vertical center distance
           0.90, // >0, upper radius
           0.40, // >0, lower radius
           0.80, //>2*border radius, length in z-
           0.10, // border radius,
             // max. = min(lower radius, upper radius)!
           0 // Merge_On: 0 or 1
        ) // ------------------------------------
        texture{ pigment{ color rgb <1,1,1>}
                 finish { phong 0.1}
               } // end texture
        scale <1,1,1>
        rotate <0,0,0>
        translate < 0, 1, 0>
      } // end of object --------------------
