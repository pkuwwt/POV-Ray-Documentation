sphere = sphère
syntaxe générale : 
sphere{ <xM,yM,zM>, r
        texture{ ... ... }
      }
Dans ce cas "<xM,yM,zM>" sont les coordonnées du centre et "r" est le rayon.
Voici les exemples : 
sphere{< 0,0, 0>,1.0 texture{...}}
sphere{<-1,2,-3>,1.0 texture{...}}
sphere{< 2,3, 5>,1.0 texture{...}}

Si une spère doit être déformée par "scale< , , >" pour devenir une ellipsoid.
Il est recommandé de le faire selon ce
principe de construction :
Etape 1 : Déformer la forme à l'origine des coordonnées "<0,0,0>" par "scale< , , >" et ensuite
Etape 2 : Déplacer la forme à sa position finale, dans l'univers, par "translate< , , >".
Parce-que l'échelle "scale< , , >" est toujours relative par rapport au centre "<0,0,0>". Il est recommandé de le faire avant toute translation !

sphere + scale<...> = ellipsoid
syntaxe générale : 
sphere{ <xM,yM,zM>, r scale<sx,sy,sz>
        texture{ ... ... }
      }
Ici "<xM,yM,zM>" sont les sont coordonnées du centre, et "r" est le rayon, "sx,sy,sz" les facteurs de l'échelle dans les directions x-y-z.
Voici les exemples : 
sphere{<0,0,0>,1.0 scale<2,0.4,1>
      translate< 0,0, 0> texture{...}}
sphere{<0,0,0>,1.0 scale<1,1,2>
      translate<-1,2,-3> texture{...}}
sphere{<0,0,0>,1.0 scale<1,1,0.2>
      translate< 2,3, 5> texture{...}}
