I caratteri nazionali e l'Unicode

I caratteri nazionali normalemente installate ("fonts") hanno più di segni che l'ASCII-code di base. Un problema si pone con i caratteri nazionali perché POV-Ray non posso lavorare con più che l'ASCII code di base.
Per fare POV-Ray funzionare anche con i caratteri nazionali nei oggetti di "text" dobbiamo usare il comando:
global_settings { charset utf8 }


Oggetti di testo unicode
con i caratteri nazionali


Unicode con i numeri dei caratteri decimale: 
global_settings { charset utf8 }
text{
  ttf "arial.ttf",
  concat(
  chr(196),chr(203),chr(214),chr(220),
  " ",chr(8364)), 0.01, 0
  pigment{color rgb<0.5,0,0>}
  scale 0.4 translate<-0.5,0.5,0> }

Alternativo con i numeri dei caratteri esadecimale
("hex", "hexdecimal"): 
global_settings { charset utf8 }
text{
  ttf "arial.ttf",
  "\u00C4\u00CB\u00D6\u00DC \u20AC",
  0.01, 0
  pigment{color rgb<0.5,0,0>}
  scale 0.4 translate<-0.5,0.5,0> }
