torus = Torus, Ring
torus = Torus, Ring, Reifen
Allgemeiner Syntax: 
torus{ rmajor, rminor
        texture{ ... }
        } //----------
Hier ist rmajor der große Radius
und       rminor der kleine Radius.
Der Torus liegt in der Ausgangslage in der xz-Ebene (horizontal)
mit der y-Achse als Drehachse.
Außenradius =
rgesamt = rmajor + rminor .
Innenradius =
rinnen = rmajor - rminor (Loch im Ring).
Will man eine andere Lage erreichen, so muß der Torus mittels "rotate< ,  ,  >" und "translate< , , >" in die gewünschte Position gedreht und geschoben werden.
 Die Beispiele hier: 
//--------------- der gelbe Torus in der Mitte
torus{ 1.0,0.25 scale <1,1,1>
       texture{ pigment{ color rgb<1,0.65,0>}
                finish { phong 1}
              }
       rotate<0,0,0> translate<0,0,0>
     }
//------------ der goldene Torus rechts hinten
torus{ 1.0,0.25 scale <1,3,1>
       texture{ Polished_Chrome
                pigment{ color rgb<1,0.8,0>}
                normal { bumps 0.5 scale 0.15}
                finish { phong 1}
              }
       rotate<90,0,0> translate<4,3,2>
     }
//------------- der flache silberne Ring links
torus{ 1.0,0.65 scale <1.5,0.33,1>
       texture{ Polished_Chrome
                pigment{ color rgb<1,1,0.5>}
                normal { bumps 0.5 scale 0.15}
                finish { phong 1 }
               }
       rotate<0,0,90> translate<-1,3,-2>
     }
