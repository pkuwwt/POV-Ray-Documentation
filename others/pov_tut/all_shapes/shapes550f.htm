Formes cubiques & quartiques du fichier include "shapesq.inc",
une fichier include de Alexander Enzmann

Avant de pouvoir utiliser les formes géométriques le fichier include doit être inséré dans la description de scène par :
#include "shapesq.inc"
Pour POV-Ray for Windows il y a ci quelques Insert Menu item pour cette formes à la disposition dans mon "Insert Menu Add-on".
