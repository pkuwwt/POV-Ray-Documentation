  Round_Cone - cono arrotondato
    Dichiarato in "shapes.inc"
Round_Cone
type 1 
// ---------------------------
object{
 Round_Cone(
  //point A, radius A:
  <0,0,0>,0.70,
  // point B, radius B:
  <0,1.2,0>, 0.40,
  0.15, // rounding radius,
  0)// merge

 pigment{
 color rgb<0.75,0.35,1>}

 scale<1,1,1>
 rotate<0,0,0>
 translate<0,0.0,0>
} //--------------------------
 Round_Cone2
type 2 
// ---------------------------
object{
 Round_Cone2(
  //point A, radius A:
  <0,0,0>,0.70,
  // point B, radius B:
  <0,1,0>, 0.40,
  // merge on
  0)//

 pigment{
 color rgb<0.75,0.35,1>}

 scale<1,1,1>
 rotate<0,0,0>
 translate<0,0.0,0>
} //--------------------------
 Round_Cone3
type 3 
// ---------------------------
object{
 Round_Cone3(
  //point A, radius A:
  <0,0,0>,0.70,
  // point B, radius B:
  <0,0.8,0>, 0.40,
  // merge on
  0)//

 pigment{
 color rgb<0.75,0.35,1>}

 scale<1,1,1>
 rotate<0,0,0>
 translate<0,0.0,0>
} //--------------------------
Annotazione:
Tutti questi "Round_Cone" sono dichiarati come macro in unioni.
Round_Cone é una unione di un cono e due tori.
Round_Cone2 e Round_Cone3 sono unioni di un cono e due sfere.
Per questo, si usiamo testure ou materiale trasparenti,
dobbiamo usare "merge" con attivare il merge flag "1".

I tre forme sono dichiarate nel file include "shapes.inc".
