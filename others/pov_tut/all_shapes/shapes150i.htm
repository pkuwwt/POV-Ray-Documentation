box = parallelepipedo rettangolo,
cuboide, scatola
Sintassi generale: 
box{ <x1,y1,z1>,<x2,y2,z2>
     texture{ ... ... }
     rotate<0,0,0>
     translate<0,0,0>
   } ---------------------
Qui <x1,y1,z1> e <x2,y2,z2> sono le coordinate di due angoli diametrali di fronte.
L'esempio qui: 
box{ <2,0,1>, <5,2,6>
     texture{ ... ... }
   } // -----------------
