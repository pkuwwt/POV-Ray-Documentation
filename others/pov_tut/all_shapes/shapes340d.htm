// ---------------------------------------
object{
 //Round_Cylinder(A,B,Radius,EdgeRadius,Merge)
 Round_Cylinder(<0,0,0>,<0,1,0>,0.50,0.20,0)
 texture{
   pigment{ color rgb<1,0.1,0.35>}
   finish { phong 1}
 } // end of texture
 scale<1,1,1>
 rotate<0,0,0>
 translate<0,0,0>
} //---------------------------------------

Achtung:
Edge_Radius > Distanz(A;B).
Radius des Cylinders > Edge_Radius.
