box = rectangular cuboid, rectangular parallelepiped
general syntax: 
box{<x1,y1,z1>,<x2,y2,z2>
     texture{ ... ... }
    }
Here are <x1,y1,z1> und <x2,y2,z2> the coordinates of two diametrically opposed points.
Example here: 
box{ <2,0,1>,<5,2,6>
     texture{ ... }
   } //--------------
