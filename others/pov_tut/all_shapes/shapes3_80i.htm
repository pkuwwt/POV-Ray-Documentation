Archivio esempio ".txt" o ".pov"
Round_Cylinder_Tube
Un cilindro con estremità arrotondate, piene o aperti!
Sintassi generale: 
object{ Round_Cylinder_Tube(
                       Point_A,
                       Point_B,
                       R_major,
                       R_minor,
                       Filled,
                       Merge_On
                    ) //----------------
         texture{ ... }
      } // end of object ---------------
Point_A = punto iniziale A,
Point_B = punto finale B,
R_major = raggio majore, (cilindro),
R_minor = raggio del bordo,,
Filled: 1 = pieno; 0 = aperto
Merge_On 0 = union, 1 = merge.
Esempio: 
#include "shapes3.inc"
object { Round_Cylinder_Tube(
            <-1,-0.2,-0.2>, // start
            <1.0,0.9,-0.3>, // end
            0.95, // major radius
            0.12, // border radius
            0, //  1 = filled; 0 = open
            0  //  Merge_On,
        ) // ------------------------------
        texture{ pigment{ color rgb<1,1,1>}
                 finish { phong 0.1}
               } // end texture
        scale <1,1,1>
        rotate <0,0,0>
        translate < 0, 0.9, 0>
      } // end of object ------------------
