box = Quader, Schachtel
box = Quader,
rechtwinkliges Kuboid
Allgemeiner Syntax: 
box{ <x1,y1,z1>, <x2,y2,z2>
     texture{ ... ... }
     rotate<0,0,0>
     translate<0,0,0>
   } ---------------------
Hierbei sind <x1,y1,z1> und <x2,y2,z2> die Koordinaten von einander räumlich gegenüberliegenden Eckpunkten.
Das Beispiel hier: 
box{ <2,0,1>, <5,2,6>
     texture{ ... ... }
   } // -----------------
