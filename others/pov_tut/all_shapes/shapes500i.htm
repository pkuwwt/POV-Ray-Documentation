Polyhedra regulari del include file "shapes2.inc"

Prima che si puo usare le forme geometriche seguente si deve inserire il include file corrispondente nel testo della scena con:
#include "shapes2.inc".
 

Fila in alto row di sinistra:
Rhomboid
Hexagon = Esagono, prisma regulare con 6 lati
sphere{<0,0,0>,1} (Sfera normata a paragone)
Pyramid = Pyramide, lungezza del lato = lungezza della base
Pyramid2 = Pyramide con 45 gradi
Fila in basso di sinistra:
Tetrahedron = Tetraedro, una piramide con base triangolare. (qui con "rotate<0,180,0>"),
Octahedron = Ottaedro, una piramide doppio, con 8 lati
Dodecahedron = Dodecaedro, 12 lati
Icosahedron = Icosaedro, 20 lati
Octahedron s.o.
Questi oggeti geometici regolari sono defininito in simmetria a <0,0,0>.
Un esempio come usarle con la parola chiave "object": 
object{Icosahedron
       texture{....}
       translate<...> }
