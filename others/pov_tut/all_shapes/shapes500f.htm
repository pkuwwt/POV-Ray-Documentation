Polyèdres réguliers du fichier include "shapes2.inc"

Avant de pouvoir utiliser les formes géométriques le fichier include doit être inséré dans la description de scène par :
#include "shapes2.inc"  
 

Polyèdres réguliers :

Ligne inférieure à partir de la gauche :
Tetrahedron = Tétraèdre, pyramide avec des angles triangulaires (fait avec "rotate<0,180,0>"),
Octahedron = Oktaèdre, double pyramides, 8 côtés
Dodecahedron = Dodécaèdre, 12 côtés
Icosahedron = Icosaèdre, 20 côtés
Octahedron s.o.
Ligne supérieure à partir de la gauche :
Rhomboide
Hexagon = prisme régulier à six côtés
sphere{<0,0,0>,1} (sphere unité afin de comparer )
Pyramid = Pyramide, côté de longueur = largeur de la base
Pyramid2 = Pyramide de 45 degrés
Ces formes géométriques sont définies en tant que symetric to <0,0,0>.
Exemple pour montrer comment les utiliser comme objet : 
object{Icosahedron
       texture{....}
       translate<...> }
