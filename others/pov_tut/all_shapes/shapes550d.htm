Cubic & Quartic Formen aus der Include-Datei "shapesq.inc",
einer Include-Datei von Alexander Enzmann

Vor Verwendung der im folgenden beschriebenen Grundfiguren muß die jeweilige POV-Ray-Include-Datei im Programmtext eingebunden werden. Dies geschieht durch:
#include "shapesq.inc"
Für POV-Ray for Windows gibt es auch Insert Menu items für diese Formen. Sie sind in meinem "Insert Menu Add-on" zur Verfügung.


object{ Glob_5 //---------------------
        sturm
        texture{
           pigment{ color rgb<1,.9,.7>}
           finish { phong 1}
               }
        rotate <0,0,-90>
        scale <1,1,1>
        translate <0,1,0>
       } // ------------- end of object
