LeoCAD ist ein CAD-Font-end für LDraw (Windows + Linux).
Homepage von LeoCAD (en): www.leocad.org

Import- und Export-Dateiformate (load/save):
.lcd - LeoCAD Projekte
.dat - altes LDraw Format
.ldr - Einteilige LDraw-Datei( Im-/Export auch in LEGO Digital Designer ® ! )
.mpd - Multi-part LDraw-Datei
Export-Dateiformate:
.3ds - 3D Studio
.html - HTML
.pov + .inc - POV-Ray - Szenen- und Include-Dateien
.obj - Wavefront object file

Beispiel LGEO POV Library

Mit LeoCAD ist man dem besten Weg um virtuelle LEGO-Modelle im LDraw zum POV-Ray-Format zu konvertieren und sie dann von POV-Ray darstellen zu lassen. Dazu sollte man allerdings unbedingt dieLGEO POV Library verwenden.
Vergewissern Sie sich, dass Sie die neueste Version dieser Library (12-2008) besitzen.
(Download bei www.digitalbricks.org/lgeo.html).
Dieser Satz von Include-Dateien für POV-Ray dient dazu die meisten LDraw-Bausteine, welche aus oft etwas sehr eckigen Dreiecksnetzen bestehen, durch POV-Ray Grundformen zu ersetzen um dadurch Bausteine mit perfekt gerundeten realistischeren Kanten zu erhalten.
Hierzu lädt man sein Modell in LeoCAD und wählt "File" -> "Export" -> "POV-Ray".
LeoCAD arbeitet auch mit POV-Ray3.7 beta zusammen.
