LeoCAD est un logiciel DAO (CAD) pour LDraw (Windows + Linux).
Le site de LeoCAD (en) : www.leocad.org

Import and export file formats (load/save):
.lcd - LeoCAD Projects
.dat - old LDraw format
.ldr - single-part LDraw file ( im-/export also in LEGO Digital Designer ® ! )
.mpd - multi-part LDraw file
Export file formats:
.3ds - 3D Studio
.html - HTML
.pov + .inc - POV-Ray - scene and include files
.obj - Wavefront object file
Exemple LGEO POV Library
Exemple LGEO POV Library

Avec LeoCAD nous avons la méthode meilleure pour convertir modèles LEGO virtuels en LDraw dans le format de POV-Ray et faire des images en POV-Ray par utiliser la LGEO POV Library.
On doit s'assurer que on a la version plus actuelle de ces fichiers (12-2008)
(Téléchargement à www.digitalbricks.org/lgeo.html).
C'est une série de fichier include pour POV-Ray pour remplacer les réticule de triangles de LDraw par des objets primitives de POV-Ray pour obtenir des formes complètement arrondis.
Chargez votre modèle en LeoCAD et chosirez File -> Export -> POV-Ray .
LeoCAD marche aussi avec POV-Ray 3.7 beta.
