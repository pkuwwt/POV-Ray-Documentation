LDraw™
- un software di base CAD gratuito
(Windows Macintosh Linux) è un pacco de Font-Ends e Add-ons comme LeoCAD, MLCad, L3Lab, L3PAO, LDview, LPub.
Homepage di LDraw: www.LDraw.org

LDraw™ è un standard apperto (open standard) per programme CAD per LEGO® con questo il è possibile di creare modelli et scene LEGO® virtuali. Si può usarle per documentare modelli costruito fisicamente, creare istruzioni per il montaggio comme LEGO®, creare immagini 3D fotorealistiche di modelli virtuali e anche fare delle animazioni.
La lista delle possibilità è senza fine. Ma in modo meglio que con i mattoncini LEGO® reali, non si è limitato della quantità limitata degli pezzi e colori, in LDraw niente è impossibile.

Per download (Windows Macintosh Linux)
de pacco LDraw™ vede qui:
www.LDraw.org, vede qui: "Get Started 
Qui si trova anche una descrizione intensiva e istruzioni per l'installazione in inglese degli componenti di LDraw.

La versione Windows usato per mie lavori qui è "LDraw_setup_2006q3_full_J.exe"
Questa versione funziona senza problemi anche con Windows Vista, se s'installa il directory LDraw in C:\LDraw invece di C:\Program files\LDraw.

La maniera meglio per convertire modelli virtuali LEGO di formato LDraw in formato POV-Ray è quella con LeoCAD con il uso di LGEO POV Library. La versione più nuovo di questa library (12-2008) si può download a www.digitalbricks.org/lgeo.html.
Questo è un set di include file per POV-Ray per sostituire tutti i reti di triangoli (mesh) di LDraw con "forme primitive" di POV-Ray per ottenere forme più realistiche con i spigoli smussati.
Vede mio capitolo LDraw + LeoCAD per più di dettagli.
