LeoCAD è un CAD font-end per LDraw (Windows + Linux).
Homepage di LeoCAD (en): www.leocad.org

Import e export formate di file (load/save):
.lcd - LeoCAD Projects
.dat - old LDraw format
.ldr - single-part LDraw file ( im-/export anche in LEGO Digital Designer ® ! )
.mpd - multi-part LDraw file
Export file formats:
.3ds - 3D Studio
.html - HTML
.pov + .inc - POV-Ray - scene et include files
.obj - Wavefront object file
Esempio LGEO POV Library
Esempio LGEO POV Library

Con LeoCAD abbiamo il methodo meglio per convertire Modelli virtuali LEGO di formato LDraw in formato POV-Ray et poi fare immagini 3D con POV-Ray con l'uso di LGEO POV Library.
È importante di usare la versione più nuovo di questa library (12-2008).
(Download di www.digitalbricks.org/lgeo.html).
Questo è un set di include file per POV-Ray per sostituire tutti i reti di triangoli (mesh) di LDraw con "forme primitive" di POV-Ray per ottenere forme più realistiche con i spigoli smussati.
Si deve solo carricare il modello in LeoCAD e scegliere File -> Export -> POV-Ray .
LeoCAD funziona senza problemi anche con POV-Ray3.7 beta.
