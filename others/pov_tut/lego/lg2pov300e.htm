LEGO® Model Sizes & Grid

The Dimensions of Real and Virtual LEGO® Elements.


Real dimensions of LEGO® elements:


LEGO-brick 1x1 with one stud:

Grid width (Distance of studs):
Length and Width = 8 mm.

Width of the brick:
ca. 7.8 mm.
(Between two bricks there is a small gab the make it easier to stick the bricks besides of each others!)

Brick height (grid height): 9,6 mm,  
Studs height: 1.6 mmm.

LEGO plate height = 1/3 of bricks height

Original LEGO Dimensions.

Virtual LEGO®-Elements in LDraw u. LeoGEO:

In LDraw and LeoGEO a scale of 1 unit = 10mm is used.
Further details of the dimensions we can find listed in the file LGEO\lg\lg_defs.inc.
