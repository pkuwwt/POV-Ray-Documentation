Dampf, Rauch, Wolken
Verwendung von "scattering media"
zur Simulation von Dampf, Rauch und Wolken.
Damf und Rauch.

Zur Simulation von Dampf- oder Rauchwolken kann man ein "scattering object media" mit einer verwirbelten (mit 'turbulence'!) "density map" verwenden - falls nötig kann dies mit "emission" verknüpft werden. Das verwendete Container-Objekt sollte voll durchsichtig sein und das Schlüsselwort hollow besitzen. (dies erlaubt media-Effekte innerhalb!). 

pigment{ rgbt 1 }    or
pigment{ color rgbt<1,1,1,1>}
Um ein Abschneiden des turbulierten "media" zu vermeiden, muss man die Größe des Container-Objekts entsprechend erhöhen.

Das erste Bild der rechten Spalte demonstriert die Konturen des Container-Objekts (große Kugel) und die Form von "media" mit der "density map" aber ohne jegliche Turbulenz.
Container-Objekt und density map ohne Turbulenz
Rauch und Dampf mit "media"
Wenn man im folgenden Text die Phrase "emission 0.6" herauskommentiert, erhält man dunkleren Rauch. Mit "emission" kann man den Rauch zu weißem Wasserdampf aufhellen.
//-----------------------------------------------
sphere{ <0,0,0>, 1.5  // media container shape
        pigment{ rgbt 1 }
        hollow

 interior{ //---------------------
    media{ method 3
           emission <1,1,1>*1
           scattering{ 1, // Type
              <1,1,1>*5.00
            } // end scattering
           density{ spherical
                    scale 1
                    turbulence 0.85
                    color_map {
                    [0.00 rgb 0]
                    [0.05 rgb 0]
                    [0.20 rgb 0.2]
                    [0.30 rgb 0.6]
                    [0.40 rgb 1]
                    [1.00 rgb 1]
                   } // end color_map
           } // end of density
           samples 20 // higher = more precise

     } // end of media --------------------------
  } // end of interior
} //---------------------------------------------
