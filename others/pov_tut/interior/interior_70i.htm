Vapore, Fumo, Nuvole
L'applicazione di "scattering media"
per la simulazion di vapore, di fumo o di nuvole.
Vapore, Fumo, Nuvole.

Per simulare nuvole di vapore o di fumo, possiamo usare "scattering object media" con una "density" con "turbulence" e - se necessario - combinata con"emission".
L'oggetto di container usato (qui: una sfera) deve essere chiaro e con la parola chiave "hollow" . (Per admettre gli effetti di quot;mediaquot; all'interno dell'oggetto!). 

pigment{ rgbt 1 }    or
pigment{ color rgbt<1,1,1,1>}
Per evitare troncare la media turbolenta dobbiamo ingrandire l'oggetto di container.

La prima immagine nella colonna a destra fa vedere i contorni del oggetto di container (la sfera grande) e la forma di media con "density map" ma senza alcuna turbolenza.
L'oggetto di container e "density map" senza turbolenza.
Vapore, Fumo, Nuvole con "media"
Senza la frase "emission 0.6", otteniamo una versione più scuro per fumo. Con "emission" possiamo schiarire il fumo a vapore d'aqua, fumo bianco.
//-----------------------------------------------
sphere{ <0,0,0>, 1.5  // media container shape
        pigment{ rgbt 1 }
        hollow

 interior{ //---------------------
    media{ method 3
           emission <1,1,1>*1
           scattering{ 1, // Type
              <1,1,1>*5.00
            } // end scattering
           density{ spherical
                    scale 1
                    turbulence 0.85
                    color_map {
                    [0.00 rgb 0]
                    [0.05 rgb 0]
                    [0.20 rgb 0.2]
                    [0.30 rgb 0.6]
                    [0.40 rgb 1]
                    [1.00 rgb 1]
                   } // end color_map
           } // end of density
           samples 20 // higher = more precise

     } // end of media --------------------------
  } // end of interior
} //---------------------------------------------
