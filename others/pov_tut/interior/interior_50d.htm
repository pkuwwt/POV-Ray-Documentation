Licht durchs Fenster
mit Scattering Object Media
Die Verwendung "media" und "scattering" Schlüsselworte
für lokale atmospherische Effekte
innerhalb von Objekten.
Lichtstrahlen in streuendem 'media'

Sonnenlicht, das in einen Raum scheint.
Lokaler Nebel mit scattering media:
( scattering = lichtstreuend )
Um ein lokales lichtstreuendes atmosphärisches Medium für Nebel in einem Raum zu erzeugen, setzt man "media" mit einen begrenzenden Container ein:


 
Der Raum gefüllt mit Nebel:
box{
   <0,0,0>, <R_x, R_y, R_z>
   pigment { rgbt 1 } // clear
   hollow
   interior{
     media{
       scattering{ 1, 0.17 extinction 0.01}
       samples 10, 20 // min, max
     } // end media
   } // end interior
 } // end box
//----------------------------------------
Dies sieht noch ziemlich grob aus!
Man erkennt die Schichten der einzelnen "samples"!
Box mit streuendem 'media' füllt einen Raum mit Nebel:
box{
   <0,0,0>, <R_x, R_y, R_z>
   pigment { rgbt 1 } // clear
   hollow
   interior{
     media{
       scattering{ 1, 0.17 extinction 0.01}
       samples 100, 500  // min, max
     } // end media
   } // end interior
 } // end box
//----------------------------------------
Mit erhöhter Sample-Anzahl!
