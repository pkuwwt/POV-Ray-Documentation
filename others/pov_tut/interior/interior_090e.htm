Problems with overlapping object media.

The Solution:
Correct overlapping media in object.

Overlapping Object Media
If we have two overlapping object media in scene POV-Ray will have a problem to sample both correctly. The overlapping aerea (intersection of the media containers) will show partly the contures of the shapes which are containing media.

Two overlapping media in a collective object.

How to Overlapp Object Media correctly:
Here we have to define a collective shape of all the containers of the overlapping media by "merge" ( not with "union"! ). Here we're combining alle containers in one new object without any media first. This new object we have to give an interior containing all the media elements transformed like the according shapes before!
Note: an object can have in it's "interior{}" statement as many "media{}" statements as you like, they were applied just like in a csg union!
