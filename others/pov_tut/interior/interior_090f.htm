"object media" avec chevauchement
Se nous avons en scène deux "object media" que se chevauche, POV-Ray a des problèmes avec sampler correctement tous le deux. Dans la zone de chevauchement ( L'intersection des conteneurs de "media" ) on peut voir partiellement les contours de les formes que contenissent les "media".

Deux "object media" avec chevauchement.

Comment peut on chevaucher des "object media" correctement :
Ici nous devons definir avec "merge" ( non avec "union" ! ) un objet nouveau d'une forme collective de tous les conteneurs, comme un premier pas sans aucun "media". Ensuite nous devons ajouter à cet objet nouveau un "interior" que contenit tous les éléments de "media" transformés comme avant les conteneurs correspondants !
Note : un objet peut avoir dans son déclaration de "interior{}" tant de "media{}" que vous voulez, ils sont appliqués de la même façon d'une union csg!
