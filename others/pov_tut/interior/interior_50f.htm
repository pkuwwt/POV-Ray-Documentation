Lumière par la Fenêtre
avec Scattering Object Media
Utiliser les mots clè "media" et "scattering"
pour des effets atmosphériques localisés
à l'intérieur des objets.

Rayons de lumière avec "scattering media">

Le soleil donner dans la chambre.
Un brouillard localisé avec "scattering media" :
( scattering = diffusion de lumière dans toutes les directions )
Pour produire un médium atmosphérique diffusant localisé en forme de brouillard dans une chambre, on doit appliquer "media" avec un conteneur limité :


 
Le chambre rempli avec brouillard :
box{
   <0,0,0>, <R_x, R_y, R_z>
   pigment { rgbt 1 } // clear
   hollow
   interior{
     media{
       scattering{ 1, 0.17 extinction 0.01}
       samples 10, 20 // min, max
     } // end media
   } // end interior
 } // end box
//----------------------------------------
Cala fait un peut à désespérer ! Trop imprécis !
On peut voir le plans des "samples" séparé!
Une "box" de "Scattering media" remplit le chambre avec brume :
box{
   <0,0,0>, <R_x, R_y, R_z>
   pigment { rgbt 1 } // clear
   hollow
   interior{
     media{
       scattering{ 1, 0.17 extinction 0.01}
       samples 100, 500  // min, max
     } // end media
   } // end interior
 } // end box
//----------------------------------------
Avec un nombre de "samples" plus augmenté !
Pour terminer ajutons une porte :
Description de la scène pour POV-Ray:
