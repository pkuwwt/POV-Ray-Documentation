L'application de "scattering media"
pour la simulation de vapeur, de fumée ou des nuages.
Vapeur, Fumée, Nuages.

Pour simuler nuages de vapeur or de fumée, nous pouvons utiliser "scattering object media" avec une densité avec turbulence et - si nécessaire - combiné avec emission.
L'objet de conteneur utilisé être transparente et avec le mot clè "hollow". (Pour admettre des effets de média dans l'intèrieur !). 

pigment{ rgbt 1 }    or
pigment{ color rgbt<1,1,1,1>}
Pour éviter couper le média turbulente nous devons agrandir l'objet conteneur.

La première image dans la colonne à droit fait voir les contoures du objet conteneur (la grande sphère) et la forme du média avec "density map" mais sans aucune turbulence.
L'objet conteneur et "density map" sans turbulence
Vapeur, Fumée, Nuages par "media"
Sans la phrase "emission 0.6", nous gagnon une version plus sombre pour fumée,
avec "emission" nous pouvons éclaircir la fumée à vapeur d'eau blanc.
//-----------------------------------------------
sphere{ <0,0,0>, 1.5  // media container shape
        pigment{ rgbt 1 }
        hollow

 interior{ //---------------------
    media{ method 3
           emission <1,1,1>*1
           scattering{ 1, // Type
              <1,1,1>*5.00
            } // end scattering
           density{ spherical
                    scale 1
                    turbulence 0.85
                    color_map {
                    [0.00 rgb 0]
                    [0.05 rgb 0]
                    [0.20 rgb 0.2]
                    [0.30 rgb 0.6]
                    [0.40 rgb 1]
                    [1.00 rgb 1]
                   } // end color_map
           } // end of density
           samples 20 // higher = more precise

     } // end of media --------------------------
  } // end of interior
} //---------------------------------------------
