Une Épée de Lumière
par Emitting Media
L'application du mot clé "media"
pour la simulation des objets lumineux.

Une épée de lumière avec média emission.

Avec média de type émission nous pouvons we simuler objets incandescent. L'objet de conteneur usé doit être claire et avec le mot clè "hollow" (Pour admettre les effets de média et de " fog " a l'intèrieur !). 

pigment{ color Clear }    or
pigment{ color rgbf<1,1,1,1>}
Note: Le couleur de la lumière "emis" est spécifiée après le mot clè " emission ". Mais le média même ne peut pas vraiment "émettre" lumière - il veut dire seulement que le média est visible aussi sans aucune illumination externe !
Quand nous voulons un média que brille et que fait de lombre etc., nous devons ajuter une source de lumière au centre !


Le noyau incandescent
Le noyau de l'épée de lumière :
#declare Blade_Center =
object{
  Round_Cylinder(<0,0,0>,<0,1,0>,
                  0.02, 0.01, 1)

     pigment{ color rgbf<1,1,1,1>}
     // the container should be clear
     hollow
  interior{
      media{
             emission <1.0,0.1,0.5>*7
          } // end media
         } // end interior
  } // end object
//----------------------------------------
L'aura incandescent
Une aura incandescent on peut faire avec la même
méthode mais avec un rayon plus grand.
Après ajuter une source de lumière et une poignée
l'épée est complete. 
union{
 light_source{<0, 0, 0>
               color rgb <0.5,0.0,0.3>*0.75
               area_light
               <0, 0.1, 0> <0, 0.9, 0> //
               1, 5                    //
             } //---------------------------
 object{ Blade_Center }
        //----------------------------------
 object{ Blade_Aura }
        //----------------------------------
 // + handle see scene description !
}// end of union ---------------------------
Description de la scène pour POV-Ray:
"Light_Sword_1.txt" ou "Light_Sword_1.pov"
