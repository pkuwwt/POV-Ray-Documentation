Il Luce attraverso della Finestra
con Scattering Object Media
L'usa della parola chiave "media" et "scattering"
pour effetti atmosferici localizzati
all'interno di oggetti.

Raggie di luce con "scattering media">

La luce del sole penetra in una stanza.
Una nebbia localizzata con "scattering media" :
( scattering = diffusione di luce in tutte dans tutte le direzioni )
Per produrre una medium atmospherica diffusante localizzata in forma di nebbia in una camera, si deve applicare "media" con un container limitato:


 
La camera pieno di nebbia:
box{
   <0,0,0>, <R_x, R_y, R_z>
   pigment { rgbt 1 } // clear
   hollow
   interior{
     media{
       scattering{ 1, 0.17 extinction 0.01}
       samples 10, 20 // min, max
     } // end media
   } // end interior
 } // end box
//----------------------------------------
Questo non è molto affascinante! Troppo impreciso!
Si può veder i piani seperati del "samplling"!
Un "box" di "Scattering media" riempisce la camera con una leggera foschia:
box{
   <0,0,0>, <R_x, R_y, R_z>
   pigment { rgbt 1 } // clear
   hollow
   interior{
     media{
       scattering{ 1, 0.17 extinction 0.01}
       samples 100, 500  // min, max
     } // end media
   } // end interior
 } // end box
//----------------------------------------
Con un numero di "samples" più aumentato!
