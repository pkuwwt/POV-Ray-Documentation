 Textures per POV-Tree      
Utilizza JavaScript per visualizzare le immagini a risoluzioni più elevate.

In un file di dati POV-albero di un albero di blob ('Salva come ...')
ci sono dichiarate i colori del tronco (compresi rami e ramoscelli)..
Così i colori delle foglie vengono dichiarati in questo file.
Questi valori possono essere modificati con l'apertura di questo include file nel programma POV-Tree, cambiamenti nella sezione appropriata di POV-albero e memorizza nuovamente l'albero.

Un altro metodo semplice è quello di sovrascrivere i valori base di nuove dichiarazioni.
   Per i dettagli sulla funzione di tutti i parametri per alberi BLOB TOMTREE.inc vedono la quotazione in 'tomtree.txt'.

Come modificare i colori fogliame:
L'albero blob di base come dichiarato dal
"povtree_001.inc" con colori di primavera:
#include "povtree/trees/povtree_001.inc"
// #declare NEW VALUES here!  
#include "povtree/TOMTREE-1.5.inc"
//-----------------------------------
#declare Tree_01 =
object{ TREE double_illuminate hollow}
//-----------------------------------
object{ Tree_01
        scale  4
        rotate<0, 0,0>
        translate<0, 0.00,0>
      } //---------------------------

#include "povtree/trees/povtree_001.inc"
// winter
#declare LEAVES= 1*BUNCHES;
#declare BOTTOM_COLOR_1=<0.4,0.2,0.1,0,0>;
#declare BOTTOM_COLOR_2=<0.3,0.2,0.1,0,0>;
#declare TOP_COLOR_1=<0.4,0.3,0.2,0,0>;
#declare TOP_COLOR_2=<0.4,0.3,0.2,0,0>;
#include "povtree/TOMTREE-1.5.inc"
//-----------------------------------
#declare Tree_01 =
object{ TREE double_illuminate hollow}
//-----------------------------------
// snow
#declare SUNKEN_TRUNK_COLOR=<1,1,1>*1.3;
#declare RAISED_TRUNK_COLOR=<1,1,1>*1.3;
#declare FOLIAGE_COLOR_AT_BRANCH_END=0;
#declare LEAVES=0*BUNCHES;
#include "povtree/TOMTREE-1.5.inc"
//-----------------------------------
#declare Tree_01s =
object{ TREE double_illuminate hollow}
//-----------------------------------
union{
  object{ Tree_01 }
  object{ Tree_01s scale<1.25,1,1>
          translate<0.015,0,-0.005>}
  scale  4
  rotate<0, 0,0>
  translate<0, 0.00,0>
} //---------------------------------
