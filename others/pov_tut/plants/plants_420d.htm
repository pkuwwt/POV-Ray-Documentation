 Texturen für Bäume mit POV-Tree      
Verwendet javascript um die Bilder in höherer Auflösung anzuzeigen.

In der POV-Tree-Datei für einen Blob-Baum
werden die Farben for den Stamm (einschl. Äste und Zweige) deklariert.
Auch die Blattfarben sind in dieser Datei deklariert.
Man kann diese Werte änderen in dem man die Include-Datei im Programm POV-Tree öffnet,
diese Werte im entsprechenden Abschnitt von POV-Tree ändert und den Baum erneut abspeichert.

Eine andere einfache Methode ist das Überschreiben der entsprechenden Werte
durch neue Deklarationen.
   Zu Details über die Funktion all der Parameter für Blob-Bäume mittels TOMTREE.inc, siehe die Auflisting in 'tomtree.txt'.

Ändern der Blattfarben:
Der ursprüngliche Blob-Baum wie deklariert in
"povtree_001.inc" mit Frühlingsfarben:
#include "povtree/trees/povtree_001.inc"
// #declare NEW VALUES here!  
#include "povtree/TOMTREE-1.5.inc"
//-----------------------------------
#declare Tree_01 =
object{ TREE double_illuminate hollow}
//-----------------------------------
object{ Tree_01
        scale  4
        rotate<0, 0,0>
        translate<0, 0.00,0>
      } //---------------------------

Winter- und Schneefarben:
#include "povtree/trees/povtree_001.inc"
// winter
#declare LEAVES= 1*BUNCHES;
#declare BOTTOM_COLOR_1=<0.4,0.2,0.1,0,0>;
#declare BOTTOM_COLOR_2=<0.3,0.2,0.1,0,0>;
#declare TOP_COLOR_1=<0.4,0.3,0.2,0,0>;
#declare TOP_COLOR_2=<0.4,0.3,0.2,0,0>;
#include "povtree/TOMTREE-1.5.inc"
//-----------------------------------
#declare Tree_01 =
object{ TREE double_illuminate hollow}
//-----------------------------------
// snow
#declare SUNKEN_TRUNK_COLOR=<1,1,1>*1.3;
#declare RAISED_TRUNK_COLOR=<1,1,1>*1.3;
#declare FOLIAGE_COLOR_AT_BRANCH_END=0;
#declare LEAVES=0*BUNCHES;
#include "povtree/TOMTREE-1.5.inc"
//-----------------------------------
#declare Tree_01s =
object{ TREE double_illuminate hollow}
//-----------------------------------
union{
  object{ Tree_01 }
  object{ Tree_01s scale<1.25,1,1>
          translate<0.015,0,-0.005>}
  scale  4
  rotate<0, 0,0>
  translate<0, 0.00,0>
} //---------------------------------
