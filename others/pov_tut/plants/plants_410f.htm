 POV-Tree exemples originaux

Ce sont les exemples originaux de formes d'arbres de POV-Tree.
Les textures originales sont souvent un peu pauvre (peut-être qu'ils étaient bons avec les précédentes versions de POV-Ray), mais il ya beaucoup possibilties à les changer plus réalistes - voir ma page Textures pour les arbres de POV-Tree.
Utilise JavaScript pour afficher des images en hautes résolutions. - Cliquez sur les vignettes !

Default
Birch - bouleau
Cherry - cerisier
Eucalyptus
Linden - tilleul
Maple - érable
Olive - olivier
Palm - palmier
Pine - pin
Poplar - peuplier
Spruce - épicéa
Willow - saule
