 Alberi con Arbaro      

Arbaro é un 'tree generator' libero, scritto in Java.
Autori: Moritz Moeller, Wolfram Diestel (admin).
Formati di esportazione: Gli alberi si può esportare come POV-Ray include files (mesh2 o POV-Ray primitives), AutoCAD .dxf o Wavefront .obj.
I parametri di alberi vengono salvati in un (leggibile) file .xml.
Il pacchetto include 16 diversi esempi di alberi (anche alcuni oggetti di erba-come).
Quality: Arbaro alberi sono buoni alberi di sfondo, non molto dettagliate, non per la vista close-up.
  File con 'mesh' per gli alberi possono avere dimensioni da 10 a 20 MB o superiore - quindi abbiamo bisogno di essere paziente, se vogliamo caricare il file in un editor.
. Dove trovare: Arbaro Sourceforge project page.
Arbaro funziona con la versione Java 1.4.2 o più recente.
  (Per Java: Sun Microsystems. Linux: sembra che non funziona con GNU Java.)
Installazione ed esecuzione:
Scarica il file (qui: arbaro_1_9_8.zip) e decomprimerlo in un direttorio a vostra scelta.
  Windows: Fare doppio clic sul file 'arbaro.jar' per avviare il programma.
  Linux: riga di comando dal direttorio Arbaro decompressa: java -jar arbaro_gui.jar

Create and export tree: (Creare ed esportare l'albero)
Formati di esportazione opztionali: POV-Ray primitives, AutoCAD .dxf, Wavefront .obj.

Ora POV-Ray dovrebbe pop-up e l'anteprima di rendering:
L'immagine anteprima di quaking_aspen - pioppo tremolo.

Nella directory seconda dovrebbe esserci il creato i seguenti file:
"quaking_aspen.pov" - il file di scena POV-Ray per l'anteprima in +w400 +h0600,
    Nota: non eseguire il file di scena POV-Ray "quaking_aspen.pov" con un altro rapporto di aspetto poi 2:3 ( per es. 400x600 o 200 x 300 etc.)
    (i.e. with a command line: +w400 +h0600 ), perché questo file non mantiene le proporzioni corrette!
"quaking_aspen.png" - l'immagine di anteprima.
"quaking_aspen.inc" - i dati per le maglie (meshes) di albero.
    Questo file dichiara:
    quaking_aspen_13_height - l'altezza dell'oggetto albero (~ in ft),
    quaking_aspen_13_stems - la maglia del tronco con rami e ramoscelli,
    quaking_aspen_13_leaves - la maglia delle foglie.
    '13' Questo è il valore di inizializzazione, impostato nel 'Create and export tree'-finestra!

    In 'quaking_aspen.inc' tutte le maglie sono dichiarate senza qualsiasi texture!
    Come aggiungere texture realistiche alle parti albero vedere la mia pagina: Textures per alberi Arbaro.
