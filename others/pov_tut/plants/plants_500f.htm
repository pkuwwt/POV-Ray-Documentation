 Arbres avec Arbaro      

Arbaro est d'un générateur d'arbre gratuit écrit en Java.
Auteurs : Moritz Moeller, Wolfram Diestel (admin).
Les formats d'exportation : Les arbres sont exportés par POV-Ray comme fichiers include avec POV-Ray (mesh2 ou POV-Ray primitives), ou fichiers AutoCAD .dxf ou Wavefront .obj.
Les paramètres de l'arbre sont enregistrés dans un fichier .xml, lisible par l'homme.
Le forfait comprend 16 exemples d'arbres différentes et aussi quelques objets semblable à l'herbe.
Qualité : Les arbres de POV-Tree sont bons arbres de fond, mais ne sont pas très détaillées, pas de close-up.
    Fichiers de maillage ('mesh') pour les arbres peuvent avoir des tailles de 10 à 20 Mo ou plus - nous devons donc être patients si nous voulons charger ce fichier dans un éditeur.
Où se procurer : Arbaro Sourceforge project page.
Arbaro fonctionne avec la version Java 1.4.2 ou plus récent.
  ( Java : Sun Microsystems. Linux: il semble ne pas fonctionner avec GNU Java. )
Installation et exécution :
Télécharger le fichier en fonction (ici: arbaro_1_9_8.zip) et le décompresser dans un répertoire de votre choix.
 Windows : Double-cliquez sur le fichier « arbaro.jar » pour lancer le programme.
 Linux : La ligne de commande à partir du répertoire décompressé povtree : java -jar arbaro_gui.jar

Preview de quaking_aspen

Dans le répertoire selon il devrait y avoir le créé les fichiers suivants :
"quaking_aspen.pov" - le fichier de scène POV-Ray pour le preview en +w400 +h0600,
    Note: Ne pas exécuter le fichier de scène de POV-Ray « quaking_a1spen.pov » avec un autre rapport d'aspect comme 2:3 (400x600 ou 200x300 etc.)
    ( par ex. avec la ligne de commande : +w400 +h0600 ), parce que ce fichier ne garde pas le rapport d'aspect correct !
"quaking_aspen.png" - l'image du preview.
"quaking_aspen.inc" - les données pour les mailles de l'arbre ( comme mesh2 ).
    Ce fichier déclare :
    quaking_aspen_13_height - la hauteur de l'objet de l'arbre (~ en ft),
    quaking_aspen_13_stems - le maillage de la tige avec les branches et les rameaux,
    quaking_aspen_13_leaves - le maillage des feuilles.
    '13' - C´est la valeur de départ, située dans la « Créer et exporter tree » -fenêtre!

    Dans « quaking_aspen.inc » toutes les mailles sont déclarées sans textures !
    Comment ajouter des textures réalistes sur les parties de l'arbre voir ma page : Textures pour Arbaro.
