POV-Tree
Ces fichiers sont des définitions des arbres de blob dans fichiers include
pour l'utilisation avec TOMTREE-1.5.inc par Tom Augst.
povtree_001
- Un bouleau en toutes saisons.
Définie comme une macro dans "povtree_001.inc".
Fichier include : "povtree_001.inc".
Nécessaire fichier include supplémentaire :
"TOMTREE-1.5.inc"
Fichier exemples :
"povtree_001_spring.pov"
"povtree_001_summer.pov"
"povtree_001_autumn.pov"
"povtree_001_winter.pov"
"povtree_001_snow.pov"
