Dettagli su Seven_Segment_LCD_Display.inc

- Elementi fondamentali -



Il Seven_Segment_LCD_Display
è un modellino pronto per l'animazione di un display a cristalli liquidi a sette segmenti con lo zoccolo.
Per Download di questo include file quod vide qui.
Si raccomanda usare la variabile "frame_number" per fare funzionare il contatore ...
per dettagli so animazione si può vedere Esempio animazione: 7-segment display.

Scala 1 unità = 1 mm.
Il modellino è fatto con una scala di 1 unità = 1mm per l'uso insieme con schede di circuiti elettronichi di Eagle3D.
Nota: Eagle3D di Matthias Weißer (de,en) è un POV-Ray plugin per Eagle di CadSoft Computer GmbH (de,en), un programma eccellente per progettare il lay-out di schede con circuiti elettronichi.

Reticolo dei contatti 2.54 mm = 0.1 inch.
Questo include file usa un reticolo dei contatti standard di 2.54 mm = 0.1 inch.

Orientazione of the Pins.
L'allineamento di due file di cinque contatti dipende su gli tipi di elementi
(quod vide "SSB_Type" - lista nella macro, qui controllato di "SS_Pin_Dir" )

Impronta del piede (footprint).
È anche possibile di fare una impronta del piede corrispondente in bianco sulla scheda del circuito.
Quest si può fare insieme con l'elemento ma anche indipendentemente.

             
Opzioni del Colore di Segmenti.
Il colore della cifra LCD è determinato con la variabile "ActiveLED_Texture_Number_".
0 = White
1 = Red
2 = Orange
3 = Amber
4 = Yellow
5 = Green
6 = Blue
7 = Violet
Testure libere per tutti gli Elementi
Possiamo anche definire testure altre come le testure prestabilite con dichiarare per es.: 
//------------------------------------------------------------------------- optional textures
#declare SSBase_Texture_1 = texture{ pigment{ color rgb<1,0.36,0.6>*1.2}} // outer sides
#declare SSBase_Texture_2 = texture{ pigment{ color rgb<1,1,1>*0.8}} // inner base
#declare SSBase_Texture_3 = texture{ pigment{ color rgb< 0.75, 0.5, 0.30>*0.5 }} // inner top
#declare SSB_Text_Texture = texture{ pigment{ color rgb<0.2,0,0> }} // name + manufacturer
#declare Active_LEDs_Texture = texture{ pigment{ color rgb<1,0.1,0.3>*0.6 }} //
#declare Inactive_LEDs_Texture=texture{ pigment{ color rgb<1,0.60,0.84> }} //
#declare DisplaySkin_Texture = texture{SSBase_Texture_1}   //

2 Stili delle Cifre
Possiamo decidere tra 2 stili di cifra differenti - quod vide l'immagine qui a sinista!
Nome del'Elemento e Produttore.
Si può cambiare il tipo di carattere (font) per i nomi con (p. es.) 
#declare Socket_Font_  = "arialnb.ttf"
La scritta del nome di un elemento e del produttore è attivata
con "SS_Text_On" (0 = off, 1 = on).
