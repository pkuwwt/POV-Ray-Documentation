Palmiers
Palm_1 - un palmier - trés détaillé, mais aussi trés lent.
L'algorithme pour les palmes est une modification de l'algorithme palmes par Mr. Tsutomu Higo, Japan.
Visitez son magnifique "Gallery Cyberbust" pour les détails.
Definé comme macro dans "Palm_1.inc".
Fichier include : "Palm_1.txt" ou "Palm_1.inc"
Fichier exemple : "Palm_1demo.txt" ou "Palm_1demo.pov"
exemple image

Palm_1quadro - une groupe de 4 palmiers.
Definé comme macro dans "Palm_1quadro.inc".
Fichier include : "Palm_1quadro.txt" ou "Palm_1quadro.inc"
Nécessaire fichier include complémentaire :
"Palm_1.txt" ou "Palm_1.inc"
Fichier exemple : "Palm_1quadrodemo.txt" ou "Palm_1quadrodemo.pov"
