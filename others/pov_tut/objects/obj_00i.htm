Istruzioni per l'uso

Qui si trova una piccola collezione di oggetti pronto per l'uso per POV-Ray.
Per la maggior parte questi oggetti sono definiti in un include file come un oggetto o come una macro.

Come usare questi oggetti?
Per usare gli oggetti definti in archivi include, dobbiamo copiare l'archivio include di quest'oggetto (con l'extensione ".inc") nella directory
"C:/Programmi/POV-Ray for Windows 3.6/Include/".
POV-Ray 3.6 con Windows XP:
"C:/Programmi/POV-Ray for Windows 3.6/Include".
POV-Ray 3.6 con Windows Vista:
"C:/POVRay_3.6/Include".
POV-Ray 3.7 beta con Windows Vista:
"C:/Users/MYSELF/Documents/v3.7/Include".

Alternativa: si può copiare l'include file nella directory dove si trova il testo della scena (".pov")
o ogni altra directory dichiarata come Library_Path dans "POVRAY.INI".
Si può trovare l'archivio "POVRAY.INI" nel editore di POV-Ray
con "Tools" - "Edit master POVRAY.INI".
Qui troviamo nelle righe in basso le declarazioni Library_Path come questo:
Library_Path="C:\Programmes\POV-Ray for Windows v3.6\include".
Dobbiamo aggiungere qui un altro sentiero come per esempio:
Library_Path="C:\Users\MYSELF\Documents\My_include_files".
Possiamo dichiarare qui fino a 20 Library_Path
(Non dimenticare di terminare e iniziare di nuovo POV-Ray
dopo avere aggiungere una directory nuova qui!).

Esempio:
Per usare l'oggetto Chair_s00( ... ), doppiamo copiare l'archivio Chair_s00.inc nella directory C:/Programmes/POV-Ray for Windows 3.6/include/.
Poi l'oggetto "Chair_s00(...)" e disponibile in un testo di scena per POV-Ray nel modo seguente: 

//------------------------------------------------
#include "Chair_s00.inc"
//------------------------------------------------
object{ Chair_s00 ( 0.80, // Chair_Height,
                    0.45, // Seat_Height,
                    0.225,// Chair_Half_Width
                    0.015,// Chair_Feet_Radius,
                  )  //--------------------------
         texture{ pigment{ color rgb<0.37,0.25,0.15>}
                  finish { diffuse 0.9 phong 1 }
                } // end of texture
         scale <1,1,1>
         rotate<0,0,0>
         translate< 0.00,0.00,0.00>
       } // end of "Chair_s00(...)"
//------------------------------------------------
