Mode d'emploi

Ici se trouve une collection des objets et des pièces pour POV-Ray prêt à l'usage.
D'ordinaire ils sont definis dans un où plus fichiers include comme objects où comme macro.

Comme utiliser ces objets ?
Pour utiliser les objets definés dans les fichiers include, on doit copier le fichier include de cet objet (avec l'extension ".inc") dans le registre
POV-Ray 3.6 on Windows XP:
"C:/Programmes/POV-Ray for Windows 3.6/Include".
POV-Ray 3.6 avec Windows Vista:
"C:/POVRay_3.6/Include".
POV-Ray 3.7 beta on Windows Vista:
"C:/Users/MYSELF/Documents/v3.7/Include".

Alternativement on peut copier le fichier include au lieu où se trouve le fichier du scène (".pov") ou a chaque autre lieu registré dans Library_Path listé dans fichier "POVRAY.INI".
On peut trouver le fichier "POVRAY.INI" dans editeur de POV-Ray par "Tools" - "Edit master POVRAY.INI". Ici nous trouvons dans une des lignes en bas les declarations des chemins de Library_Path comme cela:
Library_Path="C:\Programmes\POV-Ray for Windows v3.6\include".
Nous devons ajuter ici une ligne nouvelle de cette façon :
Library_Path="C:\Users\MYSELF\Documents\My_include_files".
Nous pouvons declarer ici jusquà 20 Library_Path (n'oubliez pas di terminer et commencer di nouveau POV-Ray après que vous avez ajouter ici un chemin nouveau!).

Exemple:
Pour utiliser l'objet Chair_s00( ... ), nous devons copier le fichier Chair_s00.inc dans le chemin C:/Programmes/POV-Ray for Windows 3.6/include/.
Puis l'objet "Chair_s00(...)" est disponible dans un text du scène pour POV-Ray de la manière suivante : 

//------------------------------------------------
#include "Chair_s00.inc"
//------------------------------------------------
object{ Chair_s00 ( 0.80, // Chair_Height,
                    0.45, // Seat_Height,
                    0.225,// Chair_Half_Width
                    0.015,// Chair_Feet_Radius,
                  )  //--------------------------
         texture{ pigment{ color rgb<0.37,0.25,0.15>}
                  finish { diffuse 0.9 phong 1 }
                } // end of texture
         scale <1,1,1>
         rotate<0,0,0>
         translate< 0.00,0.00,0.00>
       } // end of "Chair_s00(...)"
//------------------------------------------------
