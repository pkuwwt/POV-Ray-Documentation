These files are blob tree definitions in include files for the use with TOMTREE-1.5.inc by Tom Augst.
povtree_001
- A birch in all seasons.
Defined as a macro in "povtree_001.inc".
Include file: "povtree_001.inc".
Necessary additional include file:
"TOMTREE-1.5.inc"
Example files:
"povtree_001_spring.pov"
"povtree_001_summer.pov"
"povtree_001_autumn.pov"
