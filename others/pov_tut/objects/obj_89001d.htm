Details zu Seven_Segment_LCD_Display.inc

- Grundsätzliche Features -



Das Seven_Segment_LCD_Display
ist ein voll animierbares Modell eines 7-Segment-LCD-Displays mit Sockel.
Zum Download dieser Include-Datei siehe hier.
Verwenden Sie die Variable "frame_number" um den Zähler laufen zu lassen ...
zu Details über Animation siehe Beispiel Animation: 7-Segment-Display.

Maßstab 1 Einheit = 1 mm
Dieses Modell verwendet den Maßstab von 1 Einheit = 1 mm zum einfacheren Gebrauch zusammen mit elektronischen Platinen von Eagle3D.
Anmerkung: Eagle3D von Matthias Weißer (de,en) ist ein POV-Ray-Plugin für Eagle von CadSoft Computer GmbH (de,en), einem exzellenten Programm für das Layout von Platinen für elektronische Schaltkreise.

Pin-Raster 2.54 mm = 0.1 inch.
Diese Include-Datei benutzt ein Standard-Pin-Raster von 2.54 mm = 0.1 inch.

Pin-Orientierung.
Die Orientierung der beiden 5-Pin-Reihen hängt vom jeweiligen Element-Typ ab.
(Siehe die "SSB_Type" - Liste im Makro, dort controlliert durch "SS_Pin_Dir" )

Footprint.
Es ist auch möglich einen entsprechenen "footprint" in weiß auf der Platinen zu erzeugen.
Dies kann sowohl zusammen mit dem LCD-Element als auch unabhängig davon geschehen.

             
Die Farb-Optionen der Segmente.
Die Farbe der LCD-Ziffern wird durch die Variable "ActiveLED_Texture_Number_" bestimmt.
0 = White
1 = Red
2 = Orange
3 = Amber
4 = Yellow
5 = Green
6 = Blue
7 = Violet
Freie Texturen für alle Elemente
Man kann andere als die gegebenene Texturen deklarieren z.B. durch: 
//------------------------------------------------------------------------- optional textures
#declare SSBase_Texture_1 = texture{ pigment{ color rgb<1,0.36,0.6>*1.2}} // outer sides
#declare SSBase_Texture_2 = texture{ pigment{ color rgb<1,1,1>*0.8}} // inner base
#declare SSBase_Texture_3 = texture{ pigment{ color rgb< 0.75, 0.5, 0.30>*0.5 }} // inner top
#declare SSB_Text_Texture = texture{ pigment{ color rgb<0.2,0,0> }} // name + manufacturer
#declare Active_LEDs_Texture = texture{ pigment{ color rgb<1,0.1,0.3>*0.6 }} //
#declare Inactive_LEDs_Texture=texture{ pigment{ color rgb<1,0.60,0.84> }} //
#declare DisplaySkin_Texture = texture{SSBase_Texture_1}   //

2 Stile für Ziffern
Man kann zwischen 2 verschiedenen Stilen der Ziffern wählen: siehe Bild hier links!
Bauteilbezeichung und Hersteller.
Man kann die Schriftart (font) des Aufdrucks ändern, z.B.: 
#declare Socket_Font_  = "arialnb.ttf"
Der Aufdruck der Bauteilbezeichung und des Hersteller wird aktiviert durch
"SS_Text_On" (0 = off, 1 = on).
Für eine Liste der verfügbaren Bauteile siehe die folgende Seite!
