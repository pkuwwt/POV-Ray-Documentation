Les Détails de Seven_Segment_LCD_Display.inc

( Affichage digital à cristaux liquides en sept segments avec un culot )

- Les attributs de base -



Le Seven_Segment_LCD_Display
est un modèle prêt à l'animation d'un affichage digital à cristaux liquides en sept segments avec un culot.
Pour télécharger ce fichier include voir ici.
On peut utiliser la variable "frame_number" pour faire marcher le compteur marcher ...
pour des détails sur animation voyez Exemple animation: 7-segment display.

Échelle 1 unit = 1 mm.
Le modèle est projetée avec une échelle de 1 unité = 1mm pour l'usage avec des platines de circuits électroniques de Eagle3D.
Note: Eagle3D par Matthias Weißer (de,en) est un plugin de POV-Ray pour Eagle par CadSoft Computer GmbH (de,en), un programme excellent pour développer le schéma de montage des platines de circuits électroniques.

Quadrillage des plots 2.54 mm = 0.1 inch.
Cet fichier include utilise un quadrillage des plots standard de 2.54 mm = 0.1 inch.

Orientation des plots.
L'orientation de les deux files de plots dépend du type del'élément
(voir "SSB_Type" - listé dans la macro, ici controllé par "SS_Pin_Dir" )

ensemble L'Empreinte.
Il est aussi possible de fare un empreinte correspondant (footprint) en blanc sur la platine.
On peut faire cela ensemble avec l'élément LCD mais aussi indépendamment.

             
Les Options de Couleurs des Segments.
Le couleur de ciffre LCD est déterminé par la variable "ActiveLED_Texture_Number_".
0 = White
1 = Red
2 = Orange
3 = Amber
4 = Yellow
5 = Green
6 = Blue
7 = Violet
Textures libres pour tous les Éléments
Nous pouvons définir des autres textures par déclarer par ex. 
//------------------------------------------------------------------------- optional textures
#declare SSBase_Texture_1 = texture{ pigment{ color rgb<1,0.36,0.6>*1.2}} // outer sides
#declare SSBase_Texture_2 = texture{ pigment{ color rgb<1,1,1>*0.8}} // inner base
#declare SSBase_Texture_3 = texture{ pigment{ color rgb< 0.75, 0.5, 0.30>*0.5 }} // inner top
#declare SSB_Text_Texture = texture{ pigment{ color rgb<0.2,0,0> }} // name + manufacturer
#declare Active_LEDs_Texture = texture{ pigment{ color rgb<1,0.1,0.3>*0.6 }} //
#declare Inactive_LEDs_Texture=texture{ pigment{ color rgb<1,0.60,0.84> }} //
#declare DisplaySkin_Texture = texture{SSBase_Texture_1}   //

2 Styles pour la Chiffre LCD
Nous pouvons choisir entre deux styles différents de chiffres: voir l'image ici à gauche!
Nome d'un Élément et de Producteur.
Nous pouvons changer le caractère (font) pour le nomes par ex. avec 
#declare Socket_Font_  = "arialnb.ttf"
Les textes imprimés des nomes d'un élément et d'un producteur sont activés
avec "SS_Text_On" (0 = off, 1 = on).
Pour une liste des éléments availables, voir la page suivante!
