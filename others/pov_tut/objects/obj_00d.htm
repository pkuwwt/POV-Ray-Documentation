          Gebrauchsanleitung zur Verwendung der
Include-Dateien der Ready-Made POV-Ray Objects

Diese ist eine Sammlung von gebrauchsfertigen Objekten und Teilen für POV-Ray.
Diese sind meist in einer oder mehreren Include-Dateien als Objekte oder als Makros definiert.

Wie verwendet man diese Objekte?
Hierzu müssen die zu den jeweiligen Objekten gehörenden Include-Dateien (mit der Endung ".inc") zunächst in das Unterverzeichnis kopiert werden:
POV-Ray 3.6 on Windows XP:
"C:/Programme/POV-Ray for Windows 3.6/Include".
POV-Ray 3.6 on Windows Vista:
"C:/POVRay_3.6/Include".
POV-Ray 3.7 beta on Windows Vista:
"C:/Benutzer/MEIN_NAME/Dokumente/v3.7/Include".

Alternativ können diese Include-Dateien auch in das aktuelle Verzeichnis der Szenentexte (".pov") oder jedes andere Verzeichnis der in "POVRAY.INI" verzeichneten Library-Pfaden gelegt werden.

Die Datei "POVRAY.INI" finden Sie im POV-Ray-Editor
mittels "Tools" - "Edit master POVRAY.INI".
Dort finden Sie einer der letzten Zeilen Pfadangaben zu Bibliotheksverzeichnissen wie etwa:
Library_Path="C:\Programme\POV-Ray for Windows v3.6\include".
Man kann bis zu 20 Pfade hier definieren z.B.:
Library_Path="C:\Users\Mein_Name\Documents\My_include_files" oder
(Vergessen Sie nicht POV-Ray zu schließen und neu zu starten,
nachdem Sie hier einen neuen Pfad eingefügt haben!).

Beispiel:
Um das Objekt Chair_s00( ... ) verwenden zu können, speichern Sie zuerst Chair_s00.inc im Verzeichis C:/Programme/POV-Ray for Windows 3.6/include/
In Szenendateien von POV-Ray kann das Objekt "Chair_s00(...)" dann wie folgt aufgerufen werden: 

//------------------------------------------------
#include "Chair_s00.inc"
//------------------------------------------------
object{ Chair_s00 ( 0.80, // Chair_Height,
                    0.45, // Seat_Height,
                    0.225,// Chair_Half_Width
                    0.015,// Chair_Feet_Radius,
                  )  //--------------------------
         texture{ pigment{ color rgb<0.37,0.25,0.15>}
                  finish { diffuse 0.9 phong 1 }
                } // end of texture
         scale <1,1,1>
         rotate<0,0,0>
         translate< 0.00,0.00,0.00>
       } // end of "Chair_s00(...)"
//------------------------------------------------
