POV-Trees
Questi file sono definizioni albero di blob in archivi include per l'uso con TOMTREE-1.5.inc de Tom Augst.
povtree_001
- Una betulla in tutte le stagioni.
Definito in una macro in "povtree_001.inc".
Archivio include: "povtree_001.inc".
Necessario include file aggiuntivo:
"TOMTREE-1.5.inc"
Archivi esempio:
"povtree_001_spring.pov"
"povtree_001_summer.pov"
"povtree_001_autumn.pov"
"povtree_001_winter.pov"
"povtree_001_snow.pov"
